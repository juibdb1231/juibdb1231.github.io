语文老师的三生石与诗

语文老师戴着细框眼镜，黑瘦，面相清苦。曾经在某著名私立学校教初中语文。写得一手好字，在特殊的日子喜欢在朋友圈写诗。

初一的第一节课，语文老师习惯给全班讲一个‘三生石’的故事。大概意思可能是缘分历经三生三世，要大家珍惜。然而一个班往往很不统一，有人相信，有人把它归结于唯心主义而嗤之以鼻。语文老师从来不是嘴上随便说说，‘三生石’不但每学期提，还会写在朋友圈文案里，并且贯彻于行动中，之后会慢慢讲到。

语文老师教学严谨。每一篇文言文下面的注释，不管是不是考点，通通要求背下来。晚自习上，语文老师捧着书听写，声音飘荡在惨白的教室里，和记不住注释的学生手里惨白的听写纸相呼应。平日上课，语文老师点人回答问题的方式也十分严谨，必须是兼顾成绩好的学生和成绩差的学生。道理显而易见，好学生是表率，差学生是警钟，至于成绩中等的那些人，毫无特色，不值得语文老师浪费宝贵的提问机会。并且，语文老师不会仅仅提出和课本有关的问题。譬如某学期开学的第一节课，语文老师让大家谈了一下对中华民族伟大复兴的理解。

语文老师自认经历过风浪，不曾想却在带某个实验班的时候翻了车。这个班成绩挺好，但是学生十分不让人省心。语文老师上课时，两个邻座的成绩好的学生叽叽喳喳，目中无人；青春期的男生会在听到一些放到今天的互联网上都不会因为敏感被哔掉的字眼时怪声怪气；更有甚者，竟公然忽略所要求阅读的中华经典《水浒传》，简直不可理喻。语文老师每次看到有关《水浒传》的试题，都会点名那位成绩不错，但是连扈三娘孙二娘都分不清楚，更不用说类似黑旋风李逵这种人物匹配的同学。然而该同学有n次答不上来，就有n+1次选择继续不读水浒，给语文老师留下了不太好的印象。

时间累积，语文老师忍无可忍，双手伸于胸前作交叉状（神还原某emoji），用低沉的声音怒吼：把我封杀了吧。

不过很快，语文老师被封杀失败，继续任教直至该班级初中毕业。

语文老师坚信，语文从生活里来，也要到生活中去，在现世中做不到的归园田居，在诗里可以。语文老师没有笔名，每每使用姓名拼音首字母，朴素大方。作品不追求辞藻华丽，而是直抒胸臆，令人倍感亲切。

然而，语文老师的朋友圈不仅有诗。前面说到语文老师用行动贯彻三生石，诚不欺也。语文老师曾教过一位极优秀的学生，高考勇夺本省状元。虽然只教过她初中，但俗话说一日为师终身为父，在得知学生准备结婚后，语文老师便筹备了一场婚庆活动。活动当然不是婚礼，目的也不光是看昔日的学生披上婚纱，语文老师想做的，是借此机会把状元新娘、目前正在带的初中班学生和曾经带过的学生聚在一起，诚恳交流，相互学习。凑巧的是，曾经带过的学生正是来自被封杀班。那个班的学生人才济济，高中毕业之后有出国的，有考上清华北大的，以及考上其余院校的（语文老师还是遵循上课点人问答问题时的聚类准则，只是‘好’的标准犹在，‘差’却有些模糊，索性就把它和中等类合二为一）。

那天参加语文老师活动的来自被封杀班的学生，两男两女。女生一和男生一踏实稳重，清华大学就读，深受语文老师欣赏。男生二智商卓群但吊儿郎当，同样清华在读，剩下的女生二就是当年不读水浒的那位，既不踏实稳重也没有考上清华。语文老师和四人坐同一圆桌，落座之后，左手拉着女一右手拉着男一，把二人安排在了离自己物理距离最近，同时也是心理距离最近的位置。女二和男二见状，在圆桌空白处坐下，盼望着尽快上菜。语文老师心系教学，在席间抛出一个类似‘现在班上有学生学习积极性不高，成绩很差，该怎么办’的宏大教育命题。清华学子外加女二侃侃而谈，建言献策，语文老师频频点头。

语文老师教书育人之余，组织过多场类似的，以‘三生石’为精神内核的活动，少则两三人，多则十数人，聊凌云壮志，品四海美食。合照是惯例，语文老师总是竖大拇指，双手居多，通过手势表达出你真的真的真的真的真的很不错。活动后会发朋友圈，让曾经的加过微信的学生们都能感受到妙不可言的缘分。现如今语文老师的标准也逐步放宽了，不再只接收北大清华到朋友圈，复旦和上交的也没有问题，只是非985的普通211院校仍不入其眼。

拼搏奋发的语文老师几年前在职跳槽，目前在另一所学校当主任。拿大包这种想法太俗气，桃李满清北复交哈佛耶鲁斯坦福的语文老师想的，定是不管在哪里，都把三生石的故事讲下去，以及，在俗世中坚持写诗。

枣庄杨爷

杨爷来自山东枣庄，身高一米七好几，和我本科同院，研究生同班。

对杨爷的初印象，源于高等代数课上。当时尹老太让各班班长点人，坐在后面划水的我看到前面有一个高大的背影缓缓升起，心想这女生还真高。之后的三年几乎没有和杨爷说过话，直到大四搬到本部，听说杨爷和我申到了同一所学校的同一专业，想着一起订机票一起租房，才慢慢熟起来。

杨爷睡眠贼好。我作为一个喜欢胡思乱想的人，在飞机上很难睡得安稳。但杨爷不一样。在去往美国漫长的十几个小时飞行中，我数次转过头，看到睡得香甜的杨爷。然而打哈欠会传染，睡觉并不会，我如此近距离得感受到沉睡的气息，却依旧十分清醒。

安顿下来之后我和杨爷打算正式开学前去学校探探路。杨爷展现出了极强的认路能力。在使用谷歌地图一星期之后杨爷对路线基本熟稔于心，而路痴的我过了一个学期仍旧离不开谷歌地图。可以让我稍感安慰的是，杨爷体能和我一样差，我们从住所出发一路爬坡去学校，中途必须停下来歇一会喘口气续命。好在正式开学后从住处到教学楼点对点的shuttle bus救了我俩，真是一个伟大的交通工具。

开学后，杨爷给我们朴实无华且枯燥的课堂带来了生机。A课上，教课的中老年杨姓男子语调毫无变化且课件丑得感人。杨爷在他的课上睡眠一如既往的好，会时不时打个小盹。有段时间我一发现杨爷打瞌睡就抓起手机偷拍，甚至会吸引整排同学的注意。另一门B课上，杨爷的一个闹钟恰到好处地在Giles拖堂之际响起，仿佛某种革命的暗号。

生活中的杨爷对细节有相当的要求。至今记得第一次合作咖喱土豆鸡肉的时候，杨爷把我滚刀随意切的大块土豆改成几乎一样的小块，说是能更好的入味。杨爷给书桌仔仔细细贴了层纸，把房间收拾得简洁清爽，让我每次进去都感受到自己狗窝与之巨大的差距。当然除此之外，生活中杨爷有那么些令人迷惑的行为。比如早上在微波炉热早餐的短短一两分钟，杨爷也会关上房门。并且杨爷在用水壶接自来水烧水的时候，总是直接把水壶直接放水槽里，与底座接触的地方总是沾到水，让我很担心会不会漏电（后来证明想多了）。

说到这里还没有好好解释，为什么杨爷会被叫“爷”。身高固然是一个重要因素（曾经戏言以后男朋友身高可以参照爷的来），山东人的豪爽可能也是原因之一，但是我觉得最重要的是，杨爷是那种跟着姐走不吃亏的性情中人。据爷说曾在三岁时提着菜刀追人，且在她弟面前有着不可冒犯的威严，令其颇为害怕。很遗憾，没有见过提菜刀的爷，但是见识过在大家还没有很熟的时候，组织班上数十人浩浩荡荡杀向冰淇淋店。边吃边玩的爷。不可否认组织这种团体活动还是需要身高优势，然而爷拥有更为难得的广结善缘的能力。

第一学期，一米七好几的杨爷在花痴了一阵同为山东人的TA之后遇到一米八好几的飞哥并且开始互生好感。这其中有一个有小插曲。两人正式确定恋爱关系是在第一学期结束准备回国的前一天晚上，地点是一间airbnb的客厅里。当时我们几个人一起租了一晚，我半夜上厕所时客厅和洗手间灯都亮着。迷迷糊糊中，看到杨爷躺在沙发上，和坐在椅子上的飞哥聊得甚是投入。我无意于破坏气氛，想匆匆撤回卧室，然而飞哥叫住我让我重新把洗手间灯打开这样客厅不会太暗。就这样我在杨爷恋爱的重要节点掺了一脚。

扯远了，说回谈恋爱的杨爷。自此之后，准确讲就是第二学期，杨爷的主要活动，包括上课/自习/在路上走等等，都和飞哥组队。杨爷慢慢展现出本性中感性的、小女生的一面。但是由于我实在没有恋爱经验，写出来恐怕也词不达意，就此略过罢。

毕业之后，杨爷在纽约时代广场高档写字楼谋到了一份差事，那时的我在准备接待爸妈，工作也没搞定，所以搬家的时候就拜托爷带了一些我的行李到爷的纽约长岛市新居。待二老回国之后，我暂住在杨爷那里继续找工作。白天杨爷去上班，我专心投简历面试。杨爷爱喝粥，红豆绿豆杂粮来者不拒。如果早上杨爷来不及煮粥，我就白天煮一锅晚上喝。晚上我要么继续准备面试，要么和杨爷一起追剧。当时延禧攻略正在热播，但我在看媚者无疆。原著尺度过大，电视剧一番改编之后也很精彩。主题曲‘一生等你’成为之后KTV的必唱，男女主的感情虐得人难受。当时还觉得屈楚萧演技不错，没曾想两年后会闹出这么一番主人的大戏。

在OPT即将到期之时，我终于被康州小镇一家公司收下。但与此同时纽约一家广告公司还剩最后onsite，杨爷力劝我留下，但我最终还是决定去康州。康州和纽约离得近，坐火车就能到。我推着行李箱，用塑料袋提着蒸锅，准备打车去火车站。本来没有让杨爷送去，但我竟然把电脑落在厨房一盒保鲜膜上。杨爷发现后立刻拿着电脑坐上地铁，赶在我前面到了火车站。

工作之后我会在某个周五下班后坐Uber去火车站，再坐火车转地铁去找杨爷玩一个周末。说是玩，其实大多数时候就是吃加补觉加看剧，偶尔会去逛街。杨爷会安利她吃过的好吃的乌冬面、麻辣香锅、羊肉串、鸡蛋仔以及贵妃鸡干扯面。吃饱喝足后再回家看剧，各看各的或者一起看，一个周末就这么很快滑过。

一转眼从当初在杨爷房间找工作到现在已经快两年，杨爷的房间除我之外不少人住过。从佛罗里达远道而来的局长，来见校友的佳迪王，明总，当然还有飞哥。和飞哥稍微熟一点之后开始对骂对方是狗，有一说一飞哥体型真得像腊肠，这是后话。尽管去过杨爷那里如此多次，我只接待过杨爷一次。在康州的两个住处偏远，现在住的多伦多更是出了境，也不知道有没有机会能在这边见到杨爷。

东拉西扯

睡眠是个大问题

我的睡眠质量从小就不高。我至今都把这归咎于小时候隔壁技校昼夜不停的施工队。

据我妈回忆，为了哄婴儿时期的我睡觉，她和辛医生甚至会抱着我做蹲起运动 - 然而疲惫不堪的新手爸妈已经快累得睡着了，我还是精神抖擞哭声嘹亮。

上了幼儿园的我，毫无意外地成为了午睡困难户。中午别的小朋友睡得香甜，而我在床上莫名烦躁。

有一天我不知道从哪里弄来一个签字笔笔盖，正愉悦地玩耍着，试图熬过一个中午。但是人算不如天算，这个笔盖竟然从我上铺的床上直接飞了出去，直接飞到守着大家午睡的老师脚边。我慌了，在老师有所行动之前紧紧闭上了眼睛。

后来发生的事情我选择性遗忘了。唯一记得的是老师从我身边经过的时候我试图装睡又不得其法的由于闭太紧而不停眨巴的双眼。

上了小学的我印象中从来没有午睡。

那时候的我晚睡晚起，经常性迟到。往往是同学们已经快做完课间操，我才溜进学校。放下书包躲在楼梯某处，装作和大家一起回教室。对于这种明目张胆的违纪行为，老师也不怎么管 - 毕竟我在我那所小学成绩优异，且在政治上毫无追求（后来的区三好学生市三号学生都与我无关）。因而很幸运的，我的睡眠问题没有给这一阶段的我带来任何困扰。

上了初中的我开始寄宿生活。

午睡魔咒再度袭来。那种熟悉的，在室友熟睡后辗转反侧的感觉。要命的是初中课业强度巨大，班里学霸，以及我，甚至会把作业中午带回宿舍，抓紧时间写一点。因而在辗转反侧的时候我还多了一层心理负担，因为我不仅没有补充睡眠，也没有多写一点作业。更要命的是晚上，我试图早一点睡，以此弥补中午缺的觉，然而我的室友往往会夜聊。我的内心极其矛盾，一方面我想加入，另一方面我想尝试补觉。最后结果往往是，我的沉默会让室友误认为我睡着了，她们兴高采烈聊完天睡着之后，我依旧辗转反侧。而且，即使要翻身，我也不敢过于夸张，不然就不像是一个“睡着的人”。

就这么三年过去，我的成绩缓缓下降，脑子越来越慢，理综的拖后腿性初见端倪。

高中我换到一所公立中学，重新成为走读生。然而逃不过的，仍是想睡却睡不着的午觉。

走读生无床可睡，我也赶不回家，只能趴在桌上凑合。那时候班上有一个像永动机一样的女生给我留下很深的印象，她不管小课间大课间还是中午，永远不需要睡觉，永远在写卷子或者归纳卷子的错题集。我真是实名羡慕。与此同时我还羡慕另一帮人，他们一旦开始睡午觉，几乎就能睡着，甚至还会打呼噜。而我，没有精神做一中午卷子，也没有办法熟睡到打呼噜，就只能迷迷糊糊半睡半醒地度过午休时间。这种补充精力的过程就像给一个被针扎过的轮胎打气 - 针眼不大但是气永远充不满。

上了大学的我心态已经逐渐平和了，自我接纳的过程是如此漫长。

中午依旧睡不好 - 在下午有课，必须按点起床的情况下。睡不好就睡不好吧，反正那么多年也过来了。印象中只有一次，晚上被室友打游戏的声音吵地实在睡不着，气到裂开。后来那位室友流泪道歉，我没有接受，她也没有改。大四换寝室，蓄谋已久地和这位神仙分开了。那一年的我已经开始践行迄今为止最为简单有效的办法- 晚睡。既然我做不到在周围有人没睡觉的情况下睡着（即使她只会发出很小的声音，但是只要我听见了，或是说只要我知道有人没睡，我就很难睡着），那我就成为那个最晚睡的人。这个办法非常管用。这也得益于当时的大环境，一间六人寝，两个北京人基本不在寝室住，一位喜欢去别人寝室睡，剩下的两位作息都相当规律。总之大四的我睡眠质量竟然还不错。

之后去美帝读研，住2b1b，室友和我睡得都不早，当然我还是基本贯彻了晚睡原则，即使只有两个人，为了睡好也要成为晚睡的那个。

再后来开始工作，经历了独居-与人同住-再次独居，晚睡这个从前的战术性策略已然演变成为一种恶习。并且这个时长不知道是21天的多少倍，一时半会儿恐怕是难以扭转了。

晚睡虽然能极大程度上保证尽快睡着，但顶多只有99%的置信区间。剩下1%是令人闻风丧胆的失眠。失眠失眠，不失则已，一失惊人。一旦失眠，入睡时间往往三四点起跳，甚至能亲眼看到阳光重回窗前。最近的一次失眠是在面挂一家国内互联网公司之后，直接失眠到六点。所幸之后昏睡一上午，算是没有太消耗元气。

所幸工作之后基本上没有午睡这一环节，也就不必面临早年上学期间在学校睡不着，或者周末在家爸妈都在午睡而我无所事事这么折磨人的事情。阿弥陀佛。

不过话说回来，我不是不喜欢睡觉，只是不喜欢是午觉。周末经常睡到中午才起，也是十分酣畅淋漓。

很远也很近的过往

三七记忆

巴恩斯在《终结的感觉》里说，时间是一口深井。

这是个很有意思的隐喻。随着年岁增长，人们大抵是离井口渐远。我的意思是，你二十几岁的时候，即使你对你的志向和目标很迷茫、很不确定，你却能强烈地感受到生活本身是什么，生活中的你是什么样子，会变成怎样。后来……后来，这种不确定性越来越多，相互交叉，前后纠缠，虚假记忆日渐增加。想当初，你能记住你短暂人生的全部。后来，记忆变成了一件百衲衣。

但记忆终归不是飞机上的黑匣子，只有终结之时才会被开启。

于是在似梦非梦的时候，会想起生活给我的训诫。

比如要倾向于人心向善。就像法律上的无罪推定，没有足够的证据，就不要凭个人好恶妄加揣测。暑假在LA，深深厌恶一个购物几乎全是奢侈品以至于刷爆信用卡，并且还要借钱继续买奢侈品，平时也喜欢以自我为中心的富家女。然而也正是这个讨厌的人让我在机场没有遗失手机，否则回国后真得是很麻烦。后来我也暗自庆幸没有对她彻底疏离，以至于断了后路。

比如要克服来自内心的恐惧。恐惧源于敷衍导致的底气不足。初中在外国语学校，每周都要考英语，每周都要受打击。虽然有时给母上开玩笑，说是没有在我小时候培养一下英语语感，但是自己清楚，所谓苍天饶过谁，没有付出也没资格谈收获。托福也好GRE也罢，反正都要拿下，何苦让心里塞满因恐惧而产生的无尽烦恼。

比如相信时间。曾经恨的人，即使已经过去将近十年，恨意也很难消弭。当然这个恨是就我这个年龄而言的，没有涉及生活艰辛磨难或者国仇家恨。我原以为是我太狭隘，甚至因无法谅解而焦虑，然而时至今日，我并没有被这种恨所囚固。因此我允许这种情绪在不滋长的情况下留存，毕竟宽恕不是随意的布施。以后的事，交给时间罢了，不必强求。

“有人曾言，历史上他最喜欢的时光在于事物崩溃之际，因为那意味着新事物正在诞生。假如我们把这一观点运用到每一个个体的生活之中，还能说得通吗？在新事物正在诞生之际死亡——即使那新生的只是我们固有的自我？因为，正如一切政治和历史变革迟早会令人失望，成年大抵也是如此。人生亦然。有时候我想，生命的目的在于将我们磨得疲惫不堪，证明人生并非全然像所赞美的那样，不管这证明要多久，以此令我们对于最终的失去心甘怡然。”

窃以为，如此想法太过于悲观厌世了。最近刷完的美剧The newsroom里，那些自以为能够像堂吉诃德一样教化众生的新闻人，最后几乎惨败于现实。

但那又如何？各种幸福，来自工作的，来自生活的，不过是求仁得仁罢了。堂吉诃德悲剧性的理想主义，有人视为圭臬，有人嗤之以鼻，而包括我在内的绝大多数，则是徒有理想主义的外衣而又不得不为稻粱谋罢了。

很远也很近的过往

离癫狂那么近

“你们认为，独裁专制在德国不可能重演，对不对？”，德国中学老师Rainer向课堂的孩子们问到，是呀，在后来人看来，一个国家曾经的疯狂历史总是那么离谱和可笑，谁都不认为它会再次发生在“聪明”和“理智”的我们头上。所以，今天大多数德国人认为纳粹横行的日子不会再现；大多数日本人认为极端军国主义已经彻底远离；大多数中国人认为文化大革命的教训我们已经吸取；大多数美国人认为麦卡锡主义永难翻身……。但是，根据美国一场真实实验改编的德国电影《浪潮》却足够令我一阵毛骨悚然，后脊发凉，原来，只需要一个星期，一个星期就能产生集体性癫狂的萌芽。
　　
故事说的是一个高中组织国家体制活动周，里面有“独裁政治”、有“无政府主义”等课程，时间是一周，说是一周，每天只有一节大课，也就是五节课。电影里老师本来想要开设的是“无政府主义”但是因为种种原因被迫接下“独裁政治”的课程。讲述独裁实在是枯燥无味，他把他的课程变成了体验独裁政治的一个实验。看看他是如何进行试验的：
　　
整齐排列座位。
　　
举手发言。
　　
起立回答问题。
　　
不能称呼他的名字，而是称为先生。
　　
一起做律动，体验集体的力量。
　　
统一服装，穿白上衣、牛仔裤。
　　
为这个集体起一个名字，大家每个人说一个，最后确定为浪潮。
　　
这就是前三天的课程，有了这个开头局势就失控了，学生们制作并发布了浪潮的主页，制作了浪潮的标志，到全城去张贴，还发明了专门的手势。这些都极大地增强了浪潮的凝聚力，使得他的课堂中，座位爆满，并且有更多的人加入到浪潮当中。
　　
第五天，课程结束了，在水球比赛中，浪潮统一穿着白衬衫，制作了条幅，为他们的水球队加油助威。 在浪潮中有一个女孩，坚决反对他的做法，拒绝穿白上衣，并且印刷了很多反对浪潮的传单在学校里发放。在水球比赛的时候，她们因为没有穿白上衣，而被拒绝进入场地。她们偷偷从别的入口潜入，在发放传单的时候引发了骚乱。浪潮和非浪潮的人在水球馆内打了起来。
　　
也就是说，在一个星期里，Rainer不断向他们灌输集体主义思想，不断要求大家消除身上的个性，直到事情发展到接近失控，这时候老师也意识到了后果的可怕。

星期六，他给所有浪潮的成员发了短信，在学校的礼堂集合。竟然有二百多人，都穿着白上衣，他来了以后，请大家起立用浪潮的手势问好，然后发表演说，慷慨激昂。

此时，这堂政治课的教师赖纳·文格尔，也就是“浪潮”的元首，像一个复活的希特勒，发表了一通疯狂的政治演讲。他在台上振臂高呼，台下的学生如痴如醉。而他的那套说辞，完全是新左派的立场，反对全球化，指责“贫富不均”是真正的恐怖主义： “德国近年来每况愈下，我们是全球化的输家，但政府却告诉我们，努力干活才是摆脱危机的唯一办法。那些政治家根本就是经济动物的傀儡。失业率必须下降，我们还是出口大国，但实际上，穷者愈穷，富者愈富。恐怖活动是现今最大的威胁，而恐怖活动正是我们自己，通过散布不公正而一手造成的，当我们把自己的星球一步步推向毁灭的时候，那些富人却在一旁摩拳擦掌，建造空间站，还想从高处来欣赏这一切。”

反对者女孩并没有来，但是做在下面她的男朋友感觉到了气氛的不对，几次想要打断老师的演讲。可是根本不起作用，在老师的鼓动下，大家跟着他一起喊口号，要让浪潮席卷德国，男朋友激烈地反对，老师让把他架到前面，几个男生把他推到讲台上面，老师说我们如何对待叛徒，对待叛变者，大家都义愤填膺，要把他驱逐出去，没想到老师竟然说我们应该把他杀掉，大家都震惊了，但是可怕的是竟然没有一个人站出来反对，这时候老师才告诉大家，浪潮多么可怕！这就是“独裁政治”！我们该结束了。

总之，Rainer最后以一场欲擒故纵式的终结课程试图结束这渐渐失控的一切，他有意激化大家的情绪，让极权主义的极端表现――试图给予异端以肉体毁灭的可能展现在狂热的学生们面前，而不曾想到，一名更狂热的学生却以枪声和生命让在座的人们目瞪口呆地目睹一场悲剧，一场关于极权主义的可怕的悲剧。

影片中， Rainer面对的是群中学生，他们还没有切身的失业、生活困顿的体会，但是已经有了开始成形的人格，同样其中有人感到孤独，缺乏社会认同感，而实际上，绝大多数人都希望有认同感，害怕被异化，害怕被抛弃，希望在社会结构中得到承认。于是，集体主义这一中性的价值体系被扭曲和极端化，Rainer起初无法预料到这点，他给学生制定纪律，灌输一体化思想。这样的开始常常是美好的，因为有了归属感，一个人会更加自信，更加勇于面对生活，所以，这个过程中，我们发现，那些本身生活就不如意的孩子们表现的异常活跃，他们更缺乏这样的感觉，所以当有人振臂一呼时，对他们的感召力非常巨大。而且，这迅速的归属感产生，也会迅速产生巨大的情感冲击，想想吧，几天前还无人在乎，倍受嘲弄，突然成为一个集体中平等的一员，这样的冲击下，一群铁杆分子也随之产生，也为后来的极端埋下伏笔。
　　
而导演的辛辣之处在于，他的批判锋刃直指当下的“反全球化”运动。这是08年左右左翼知识分子和热血愤青最热衷的“正义”事业，但甘塞尔提出了警示，这可能是一场新的极权主义运动。即使对于这一桩四十年前的“反极权”教学事件，虽然甘塞尔借助它充分表达了自己的观念，但他并不愿就此止步，将整个事件不假思索地接受下来；相反，他对老师的做法表达了强烈的质疑。影片中，他设计了一个叫“ 蒂姆”的角色。他是富家子，但家庭缺少温暖，一直渴望融入集体摆脱孤独。“浪潮”成立后，他第一个响应“平等”的感召，把自己的名牌球衣球鞋全都烧掉了。通过这个集体，他感觉渺小的自我被放大了，自我价值也被提升了，他变得越来越狂热，也越来越偏执。他一改平时的懦弱形象，甚至冒着生命危险到处喷绘“浪潮”的标志。然而，当这样一个人奉献出全部身心，并从中赢得了别人的赞许，却突然发现这只是一场教学的骗局时，他一下子就崩溃了。他无法接受“浪潮”被解散，掏出手枪，打伤了一名同学后，吞枪自杀了。

一般而言，极权主义的初始阶段总是相对“温柔”，在这场实验中也有体验，对于不认同者，可以自由离开，片中两位具有独立思想的女生便陆续离开了这场运动。渐渐的，这样的温柔被抹去，一个高度统一的团体，以符号化的名称，服饰，标记，直到一个“浪潮”的打招呼的手势（让人立刻联想起那著名的“嗨，希特勒”）在向外界强势的宣示着自己的与众不同，同时开始排斥异己，如不穿白衬衣就会受到责难，同时，这个团体的攻击性越来越大，先是在全城到处贴“浪潮”标记，搞病毒式营销，接着千方百计阻止不同观点者的宣传，甚至开始产生暴力行为。这样的结果是，他们显得愈加强大，感召力也愈加强大，越来越多的人开始加入，要求分享这强大的力量，在他们眼中，这个团体的强大就是他们个人的强大，他们害怕被排除在外，他们也要融入，同时让这股力量更加强大。于是，极权主义的恐怖显现出来，它往往只需要初始的种子拨下，就可以产生自生长机制，当Rainer投下集体主义思想的第一笔“天使投资”时，都无需追加“投资”，“浪潮”运动就在一个星期让整个学校沸腾。所以，我们可以知道为什么希特勒当时依托这样的力量满足自己的欲望，因为这是一笔小支出，高回报的投资，同时，这样的反馈也不断刺激发起人的欲望。
　　
妄图彻底解构《浪潮》是一个困难的事情，因为从浪潮里可以开掘出许多有价值的内容，因为每个人心中，都有一个浪潮。

很远也很近的过往

文明的脆弱，人性的本恶——《蝇王》

我们如果去某大型机械制造厂去看一看，深入厂房和车间，就能看到那些精细奇妙的机械如何从复杂至极的图纸上变成让人瞠目结舌的样子。那些庞大的，高妙的，冷冰冰的东西经过层层捶打改造，构成了人类几乎无法驾驭的锡安之齿。你会由衷的被眼前的一切所震撼，那是想像力和创造力所能给于人的真实感动。然后，再来看这部名叫《蝇王》的电影。

电影背景是在设想出来的第三次世界大战时，一驾搭载几十个孩子的飞机因受到攻击，被迫在坠毁前把孩子们投放到一个孤岛上，为了生存和获救，起初孩子们以幼稚的民主形式集结在一起。后来，因为误解和恐怖，“蝇王”出现了，孩子们开始互相残杀，失去理智。——这个故事并不复杂。乍一看去，这只是荒岛历险故事和乌托邦故事的变种。然而《蝇王》作为一部极富盛名的电影与其本身的寓言性息息相关。

“蝇王”即苍蝇之王，源自希伯来语“Baalzebub”。在《圣经》中，被称为“万恶之首”。在电影中，蝇王最直接的意象代表是孩子们用来“祭祀野兽”的野猪头。它在高温潮湿的环境下，被无数苍蝇叮咬，发散出阵阵恶臭，极端恶心和恐怖。它一边连接着孩子们无法战胜的自然神秘主义，一边连接着孩子们因恐怖而被激发的惊人兽性。电影迷人的地方也正好在此——丰富，自然，条理清晰的使用意象。在一个相对封闭的范围内，探讨人本性中的善恶，原始兽性和现代文明的交锋，还有无法调和的茫然与绝望。或者还有更多，如果再变换点角度的话。

在孤岛这一意象被反复使用的历史中，一向习惯用来作为人类勇气和智慧的对手。它越强大，人类就更强大。在于它们的斗争中，人的精神性被放大和歌颂。通常人与孤岛的对抗多是个体行为。比如《老人与海》里的桑地亚哥，一叶扁舟是他的载体。这个孤岛的作用完全站在人这一边，是同盟军，而且是勇气再次启航的必要保证。

转过来看看《蝇王》里那些建造社会的孩子们吧。不完全脱离了由大人规则支配的社会，诸如道德，条约，义务等等，用文明世界遗留下来的理念组成了最初的民主。新批评学派的读者可能对下面一些符号感兴趣：象征着法律秩序的海螺；象征着人类科学的猪仔的眼镜；象征着希望的火堆等等。但是在这些之下，也就是这些符号和民主的根基恰恰是：拉尔夫是年龄最大，身体最强壮的孩子，他拥有获得权力的资本，他是“强”的化身。然而这个团体最后崩溃的主要矛盾，恰恰也在于杰克也是强壮者。而且在电影前半部用暖色调渲染的背后，有个更为致命的缺陷——这个“民主”没有任何监督，任何仲裁。它连形成社会公约都不够！

人类的文明并不取决于技术手段的先进。当然科学技术的进步能够很好的体验出人类文明的程度。开篇提到的大型机械厂，仿佛是人类理性和文明的绝佳佐证。但是想过没有，如果它是一个秘密军工厂，刚巧被极端分子掌握，那么这个所谓文明就是可怕的存在。我认为最能体现文明的是我们所能承载的政治。它是人性的汇总，也是人之所以为人的意义所在。个体无法提供值得借鉴的资历。

电影有几个小细节。杰克打猎后，跳起的舞蹈和单调重复的歌词；无法停止的狂欢和集体无意识；西蒙被杀时人群被诅咒一般的跳舞——这完全是宗教萌生时的原始形态。人在无法改变的环境中，产生本能的恐惧，是这一现象产生的根源。在这个时候，脆弱并且毫无效率的民主形式，其实只是一两个性格懦弱不肯妥协者的精神安慰，他们一边回想着大人世界该怎么怎么样，一边暗自神伤。而在恐怖的驱使下和群体安全感的需要，另一群人，以杰克为首的人，迅速组织起了可以谋生的队伍，一个最强者当权的专制制度。自然，在这个队伍中，个人意志凌驾一切，没有同情和民主。可是，无法否认的是，它才是在那个境地最适合的群落。善良的孩子们完成这一惊人的举动后，最大可能的忘掉了另个世界的民主。

孟子曾经说过“人之向善”如同“水之就下”，仿佛刚出生的婴儿便注定是天真无邪，毫无危害。不过对此，荀子则用一句“人之性恶，其善者伪也”一语捅破了那层伪善的窗户纸。虽然我们不愿意承认，但不管是从现实，还是内心的潜意识，似乎驱动我们的都是原始的兽性欲望。如果把荀子的话翻译成弗洛伊德语境，便相当于，被口腹之欲、生存本能和最早的性冲动操控的我们，其实才是真正的本我，是最靠近潜意识冰山底部所在。随着我们长大，为了融入社会，才慢慢构筑起了自我和超我的防线，变成了善良守法的良民。不过，一旦秩序的窗户纸被捅破，那人类又将面对如何的命运呢？

在《蝇王》中，导演给了我们一个极端的答案：失控与杀戮，即使在未成年的少年身上，也已存在。“我希望如果真的有一群孩子，面对如此情况，会有别的结果”导演布鲁克如是说，但无论如何，《蝇王》给了我们一种最接近真实的可能—— 虽说这不过是适者生存最简单的体现。只是这体现的太过讽刺：只在短短的时间里，人，被现代社会教导又被遗弃的人，转眼间就成了最野蛮的氏族部落人。就是只需要这样短暂的一瞬，人类几千年的文明就全部重建了，不存在了。文明脆弱如此，怎能不让人心惊胆颤。它随时随地就可以崩溃。因为在它里面，是千年不变的本恶人性。

在结尾，大人的世界又回来了，他们带走的是文明世界的明天——那些部落化以及可以随时部落化的孩子们。脆弱的文明有希望吗？本恶的人性有救赎吗？似乎《蝇王》永远都不会回答这个问题了。

你好，19岁

你好。

生日快乐。

在如此寂静的夜晚敲下这些文字，必然预示着不规律作息。在夜间你似乎难眠且多梦，而白天却睡得异常安稳。

一切仿佛陷入混乱。

你必须承认，你身上的许多东西，从小时候起就几乎未曾改变。九岁也好十九岁也罢，变的是年龄，而不是心
。
你还是一如既往得敏感，在意周围人的看法却永远不会说出口，宁可把所有事情烂在肚子里然后再默默咀嚼吞咽，从来没有过心思恪纯的时候，脑子里总是胡思乱想，而与此同时你却十分迟钝，譬如说在大家抒发情感的时候冷漠淡然，过去很久之后才发现，原来自己也存在着类似的情绪。

所以你显得不太合群。

你还是一如既往得内敛，或是说被动。你很少主动向父母要零花钱，觉得说不出口；你也很少主动与老师交流，觉得交流沟通和谄媚讨好的界线实在太难分清；你总是犹豫着，要不要对这个世界敞开怀抱。

你还是一如既往得挑剔，说得好听点叫追求完美，难听些则是事事苛刻。你总是轻而易举地发现周围人的缺点，然后用自认为极大的耐心来包容，就算提醒自己尺有所短寸有所长金无足赤人无完人，但你的心中多数时候是不松快的。

其实，这一切都源于安全感的缺乏。

睡觉时你必须趴着才能睡着，冬天时被子必须牢牢裹住不留空隙，你在熟悉的人面前会变成人来疯，而对陌生人你会习惯性地带上面具，出于防范，或者，免受伤害。

你害怕受到伤害。

因此你总是回避爱情。

从小到大你没有异性朋友，对男性有种莫名其妙的排斥，从不期望能获得异性的帮助。男性在你眼中是不可捉摸的，他们可能有各种性格，但是总体而言都是渴望成为主宰而使女性成为附庸，认为自己有能力保护所有人，似乎一切呵护，帮助，交流，尊重，都源于一种高高在上的男子汉的力量。虽然活了十九年，你清醒地看到在这个社会男人的地位，或者用另外一种说法，他们所扮演的角色，所承担的责任，都是远远大于女性的，你仍然不能在接收现实之后把自己调整到应有的状态。你知道，对于一个女人，有了男人的帮助会活得更轻松，是生活幸福的充分条件，可是，你就是不愿意尝试。

缘分天定，你要相信你能遇见幸福。

你是个骨子里很倔的人，并且对大部分事情持有认真的态度。然而你宁可把这两点特质归为中性，因为太倔了便是固执，冥顽不化不懂变通，太认真了，从某种意义上讲就是自我折磨，不值得。

你要慢慢学会在应该认真的地方不遗余力地竭尽全力。

比如未来的职业规划。

你想成为一名actuary，然后一路成长为优秀的chief actuary,并且通过国外的学习工作，完成国内的行业贡献。当然你也隐约明白，
一旦选择便很难回头，你无法耀眼地站在最前面，没有高得发烫的薪水，你要忍受巨大的孤独，接受不停学习考试带来的压力和挑战，还有许许多多的不了解和不理解。除此之外，你还将面对狭窄的人脉，以及无法进入主流金融界的某种尴尬。

你想要的，绝不仅仅是财富。

从暑假看了黄渤参演的话剧“活着”，关于未来太多不确定中，生成了某种难以言明的确定。人生过于短暂，所谓建功立业，不过因人而异罢了。各人有各自的万岁千秋，旁人实在毋需置喙。

况且，在你眼中的孤独，是再寻常不过的一种状态。

你永远不被别人了解，有时就连你自己也是迷茫混沌的。你每天和世界的交集，看似稠密，实际上太稀疏也太肤浅了。你的梦想与追求，奔跑路上的艰辛与坎坷，你的欢笑与泪水，喜爱与憎恶，谁又能体会，谁有义务理解？？没有罢。人是孤儿，孤零零地来，茕茕然地走，其中与他人共度，彼此慰藉，索取温暖，不过如此，凡夫俗子皆逃脱不掉。

关于未来，你不知道，也不想知道是什么样子，只是你希望，它接近你想要的样子。

足矣。

科技馆之儿童乐园做志愿杂感

儿童乐园，不仅有众多儿童，还有更众多的家长，而仅仅是滑滑梯这一简单项目（其实也不能说简单，因为该滑梯较之于普通滑梯，更长更封闭），通过它不能说世间百态尽收眼底，但起码可窥见一隅。下举数例予以说明。

某男，看似一副正人君子模样，却因为某位母亲纯属无意插了队导致自己儿子晚了一些滑下去，便气血上涌破口大骂，极尽难听之事。本认为这种毫无风度的男子可气可悲可笑可怜，但随后眼见他陪自己的宝贝儿子跑上跑下毫无怨言，上下十多二十次，耐心到极致，不由地怀疑这究竟是不是同一个人。他不是个好男人但是绝对是个好父亲。矛盾？也许吧…人的可怖的多面性再次得到证明。

某女，看似一副高知的模样，却在连问都没问过儿子的情况下与老公争执着，说不允许儿子下去。注意她用的是毫无理由可言的不行不可以不能不允许。最后她败给她老公了，只能看着她儿子下去，然后，一声毫无征兆的“天啊，他掉下去了”把我惊出了一身冷汗。试问这又不是刀山不是火海，没有让你儿子英勇就义，不过是个滑梯，是个游戏，何至于？

家长作为一个施教者，应该随时警醒，做一个规则的捍卫者而非利用者，做一个孩子潜能的挖掘者而非埋没者。我看到太多的家长不管三七二十一把孩子抱上去，然后紧紧拽住，然后大声叫下去吧下去啊，再十分留恋不舍地放手。我十分欣赏的一对年轻夫妻，蹲着问孩子你要不要去试试，得到肯定答复后便悉心指导，但绝不过多干涉，待孩子慢慢摸索着下去后，我听到愉快的笑声，看到年轻的爸爸妈妈眼中满满的爱意和真诚的赞赏。随后，小朋友又玩了几次，满足而欣喜。我问年轻夫妻小孩子几岁啦，答：三岁半。

另外，我十分困惑的一点是为什么无数家长会指着其他其他人对自己的孩子说你看其他小朋友都怎么怎么样所以你也应该怎么怎么样。这是什么逻辑，其他人和你的小孩有半毛钱关系？你既不知道他们的年龄又不知道他们的性格，难道这也是一种比较，一种无聊的，没有任何价值的虚荣？这种常见到令人麻木的现象，害了多少小孩，多少家庭？适当适量的比较会是好的，倘若滥了漫了恐怕就会让结果南辕北辙并且北得比你想象的远。

说说几个印象深刻的孩子吧。小A，在这个喧闹的地方在爸爸的陪伴下耐心摆弄着“手眼协调”（一圈铁丝和一个铁环，使铁环通过但不接触铁圈），非常安静沉稳的女孩子。小B，每次都紧跟在一个小男孩身后下滑，并且在下滑前要大声询问小男孩你到了吗我可以下去了吗数次直到得到笃定答案才敢下去，这种对别人（尤其是异性）的过度依赖在我看来相当可怕，因为毕竟最可靠的人是自己，永远都是这样。小C是个土豪，才上幼儿园大班的他已经有手机电脑和平板，并且可以和比他大十二岁的我几乎无障碍交流，尽管他一直没说清楚爸妈的工作究竟是什么并且不明白志愿者是什么意思而且坚持叫我阿姨不肯改口叫姐姐。

终于比较能明白为什么在国庆六十周年阅兵的时候胡主席在面对一堆高精尖武器装备时表情依旧严肃，却在看到孩子们笑着跑来时露出久违的笑容。孩子是祖国的未来（当然我也没有完全脱离祖国的未来这一广大的范畴），所谓后浪推前浪（不一定是长江啊，黄浦江金沙江都是这样），这是理论意义上的。当我们审视教育，却发现，未来堪忧，前浪危险。教育是一门艺术，它是分时间空间种族地域的，甚而至于是分高低贵贱的。没有普适真理，更无法统一标准。我们曾经历的，以及今后转换角色将经历的，无疑充满巨大悬念与挑战。对待现实，无法将其抬到与理想等高的位置，亦不可将其贬损到不堪。运用理性与智慧，经验与方法，塑造好自己和下一代是我的选择。

好吧，我承认自己是回不去了，否则不会站几个小时就觉得累得遭不住，不会有那么多乱七八糟的想法，更不会深更半夜码那么多字而基本无中心无重点啰啰嗦嗦不知所云。

关于军训的一些想法

军训作为每所学校必须参与的活动，只有做到规范、有序才能最大限度地体现出它的效果。
何谓规范？教育部门应该有一个专门管理学生军训的小组，使得公立学校的军训有固定的时间（如一周），固定的一批军训地点，每个地点在军训小组先期考察后，根据其配套设施及教官情况确定其价格，并上报分管军训小组其详细的训练计划（具体到每半天的训练任务），将该计划发放至每位教官、连长、营长手中，派专人监管。各军训地点的训练要求可略有差异，但总体上必须保持统一，包括所有动作（整理着装、稍息、立正、跨立、蹲下与起立、坐下与起立、脱戴帽、敬礼、齐步、正步、跑步及立定、跑步踏步、综合格斗术等）的具体要求。关于这一点的落实，可以由教育部下发专门的文件，或者录制所有军训动作标准的视频。要做到：无论在哪里军训，其成果基本相同。至于一些活动，如拔河、演讲比赛、歌咏比赛、电影放映、文艺晚会等可根据各军训地点情况，在所有规定内容完成的情况下有选择性地开展。特别想提一下阅兵式。姑且不说国庆阅兵同我们的阅兵式一个天上一个地下，光是搞阅兵式的目的就相当不纯。似乎是这么个道理：无论平时教官训练水平和实际训练效果好坏，只要憋足劲把阅兵式弄好，就一好百好。与其这样，平时还要什么训练，不过是走走过场敷衍了事，还不如从一开始就练分列式。当然，学生也有一定的责任。不是每一个人都有如此之高的思想政治觉悟，明知受苦受累还甘愿承受挑战自我。在我及大多数同学眼里，军训从根上讲就是把学分拿到，和高考啊未来人生之类的不沾边。如果某一天出来一个规定，赋予教官给每名同学打分的权利，且必须有较大区分度，这个分数要计入高考。呵，不得了，一个个准表现得特棒。打住打住，扯远了。我觉得我构想的完美军训模式还不完全是痴人说梦，具有一定的操作性。弊端是很显然的，那就是一旦教育部的人插手，就必然导致一些藏着掖着的阴暗的事发生。但作为学生，我觉得相比而言军训的规范统一管理更重要一些。

恭敬不如从命

在四中补课的倒数第二天，也就是昨天，那个教物理的老师说他是二班班主任，只教二班物理，那个教数学的老师是四班班主任，只教四班数学，那个化学老师教二班和三班，而我是一班的。

这就意味着我在未来的一年将与本年级最好的数学、物理、化学老师无缘。

心里什么滋味呢？觉得不公？当然不敢。毕竟好处不能让你一人占完了。只是觉得遗憾吧，命运他老人家又给我开了个小小的玩笑。

“命运”这个词或许缘于人们对生活的一种未知，也可以理解为一种神物。人们崇拜命运，因为命运可以在不经意间给予人苦寻无果的或有形或无形的种种；同时人们又诅咒命运，这必然由于他们受到了玩弄。不过更多的人则以命运为借口和托词，将主观发生的归咎于命运，这再普遍不过了。

记得曾经的语文试卷上有一篇古文，大意是先提出疑问，为什么好人没有好报坏人威风十足，为什么无恶不作的大贪巨恶可以长命百岁而好人一生命运多舛。作者在最后给出了淡淡的一句话：道不同，不相为谋。曾经我认为作者太敷衍也太肤浅，为何不就此深入探讨而尽停留于此。事实上我错了，与其对命运百思不得其解，自以为是地给出无数似对非对的评论和妄断，不如谦卑而退，对命运的安排“恭敬不如从命”。

原来看电视时，一遇到哪些讲述白血病患者，家庭悲惨的故事时，都会不屑一顾，觉得世界上惨的人多了去，你上了电视有什么了不起。但现在，我有些改变想法了，是，世界上惨的人太多，所有人所作的所有事加起来也是杯水车薪，可是相对于某个个体，他得到的就是全部。我们不可以因为整体必然的残缺而放弃个体得到全部的机会，不是吗？

宇宙中纯粹的事物太少太少，比如不可能绝对静止，在地球上不可能完全取得真空，事物不可能做标准自由落体运动与匀速或匀加速直线运动，不可能有纯净物因为空气中总会有灰尘……既然万事万物都无法保全，何必苛求所遇见的人和事，明知徒劳却被其左右呢。

我真得开始相信童话、神话、宗教的力量了，无论怎样，它们首先教会了人们最基本的善良，即使命运为难也虔诚而恭敬，凭借对生命的尊重与热爱来应对命运的挑战。

我不知道这算不算一种活法。
