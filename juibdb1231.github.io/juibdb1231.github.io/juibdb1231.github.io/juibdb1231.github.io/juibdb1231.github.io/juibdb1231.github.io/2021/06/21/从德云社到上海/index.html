从德云社到上海

从德云社到上海，时间距离一年，空间距离11500公里。

去年暑假腾讯出了款综艺叫德云斗笑社，时值人生空虚加上时间充裕加上出于好奇，在对德云社的了解仅限于听说过郭德纲于谦郭麒麟的基础之上，追完了所有正片节目与衍生花絮，从此在多伦多入坑。之后便以强大的自驱力疯狂恶补所有德云社的有关情况，对德云社的成立到后来艰难曲折的发展，郭德纲本人的种种过往，他的家庭他的徒弟们，德云社常演的不常演的相声，凡此种种都有了深入了解。期间通过各种渠道搜集相关资料，兢兢业业，效率惊人。有一段时间甚至每天听相声三五个小时，即使内容重复也去找同一对或者不同搭档的不同版本，最终熟练掌握传统相声三十段以上，对梁子（黑话，故事梗概的意思）熟稔于心，小曲小调说来就来，唱得虽然难听，但是自娱自乐足够了。更加难能可贵的是始终保持对一个非实用领域求知心，有一回给杨爷暴力安利完头德云社的头九之后被她一句“那头鹤呢”给问住了，于是查漏补缺，编织出了一张更加细密的德云社知识网络。

那阵子对德云社这个团体充满了无限向往，晚上睡觉前还会幻想着阴差阳错下加入这个草莽江湖。故事可能是这样的版本：本人性别男，自年幼起不爱读书，整日昏昏噩噩，惹事生非，家里也实在没有办法，在我接受完九年义务教育之后默许我进入社会接受毒打。之后我发现自己一来没文化二来没技术，只能做一些杂活，勉强填饱肚子。之后误打误撞进了德云社，一开始什么也不会，每天擦桌子扫地，给各位演员打杂，日子过得相当局促。几年过去，慢慢学到了一些相声段子，再加上人又老实，逐渐有了零星的上台机会。呆的时间久了，和这个团伙产生了感情，身上也烙下这个圈子的独有印记。然后有一天和几个差不多同时加入的社会人士一起拜师，有了各自的艺名，有了一群师兄，将来会有一群师弟。周围都是人精，虽然我老实但却不傻，有几个关系铁的哥们儿平时一起吹牛，谁有事了说一声哥几个能帮就帮，有钱出钱有力出力都没有的话出吆喝。

如果说非要从上面这段脑洞里提取出什么，那么可以看出我对一些东西的渴求，其中包括兼具个人和社会认同的职业属性，有区分度的专业技能，身边能够见着面聊得上话的同行，以及生活中无需防备愿意为其两肋插刀的朋友。至于说这些东西是否会求而不得目前尚未可知，时间会给出答案。并且要说明的是我并不是鼓吹知识无用，只是基于当时的情境，对从小到大的线性成长轨迹产生了额外的想象。家里虽然并非大富大贵有权有势，但也从没有过极度困窘生存艰难。在求学路上一次又一次的残酷筛选中跟上了大部队，和很多人一样念书考试，和一部分人一样出国，和其中一部分人一样在毕业后在国外工作。时至今日回想起来，所获取的知识带给我往前走的资本的愉悦感是远胜于实用或精神层面的纯粹愉悦感的。而德云社直接消解掉了前者，当然后者也几乎不复存在。德云社先前被社会规则压制，落于下风，最后又被同样的社会所产出的规则接纳甚至欣赏，这本身就具有一定的讽刺性。

目前德云社发展势头良好，相声综艺电视剧全面开花，团体的头部艺人阵容强大数据喜人，龙字科和鼓曲社的招新工作有条不紊开展，仿佛一代江湖霸主。然盛极必衰，合久必分，未来某一天这一切必将烟消云散。所以很想有机会在德云社走下坡路之前去贡献一份票钱，来感激这一年来给我的远程虚拟陪伴。除了拿不到票的房地产商组织的内部场，以及齁贵的专场，能近距离看到活人的地方就只有小剧场。然而我目前所想象的未来一年落脚地是上海，上海没有德云社小剧场，这令人非常遗憾。尽管现在我已经很久没有完整看过一整个节目了，德云社就仿佛那种无须多言的老伙伴，偶尔刷一下消息，看一些小片段，等着八月份斗笑社第二季上线就好，但是如果有机会去北京或者天津玩，这个票钱是一定要花的，这个约是一定要赴的。

为什么会是上海？为啥要回国？一个庸俗且不堪一击的理由是上海有很多家便利店。我非常喜欢逛超市文具店便利店等等地方，也曾幻想过有朝一日能自己开一家小超市。

抛开上述的玩笑理由，原因之首在于工作签证岌岌可危。其次，在甄嬛传里，花穗和小印子在余答应指使下试图让甄嬛慢性中毒。这像极了我现在的状态——看似每天只是感到深思倦怠，可长此以往，便会状如痴傻。再有，第一次到美国是大二暑假那年去UCLA交换，去之前曾经在上海住了几天。静安寺附近高大的梧桐给我留下了极深的印象，冥冥之中命运之手仿佛推着我去那个城市生活，尽管买不起房。

于是开始给上海的公司投递简历，态度谦卑。昨天和一个国内著名加班公司的hr聊天，hr用非常礼貌的声音问我说，自己听说国外的研究生大多都是两年，为什么我的康奈尔研究生只有一年。我一面心想说你可真厉害，都是从百度听说的吗，一面跟他耐心解释。之后该hr又问到我婚恋状况，并且主动向我介绍他家的工作时间，语气平稳，非常专业。我虽然早就在心里写满拒绝，但最后用同样礼貌平稳且专业的声音告诉他我什么时候有空面试。果然，人若是清高，是因为还不够想要。

疫情之下独居近两年，逐渐意识到自己其实是依靠周围的人活着的，我可能没有自己想像得那么不需要朋友和家人。况且只是换了一个地方工作和生活，只是有了不一样的欣喜和不一样的不如意。自由太过奢侈，能够拥有自己消费得起的自由就已经足够幸运，可问题是地广人稀并不一定意味着自由。看了一期锵锵三人行，对两句话印象很深。一句是王朔的“崩溃就是想起了以前的历次崩溃”，另一句是《地藏经》中“一切众生，未解脱者，性识无定”。或许当一游民也未为不可。

回想起上一篇文章的完工，仿佛是很久远的事。人（我）其实比想象中还要拖延。呵呵。
