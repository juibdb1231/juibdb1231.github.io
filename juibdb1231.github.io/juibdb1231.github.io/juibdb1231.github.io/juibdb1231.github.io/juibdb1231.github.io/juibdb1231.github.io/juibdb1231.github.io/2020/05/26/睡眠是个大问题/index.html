睡眠是个大问题

我的睡眠质量从小就不高。我至今都把这归咎于小时候隔壁技校昼夜不停的施工队。

据我妈回忆，为了哄婴儿时期的我睡觉，她和辛医生甚至会抱着我做蹲起运动 - 然而疲惫不堪的新手爸妈已经快累得睡着了，我还是精神抖擞哭声嘹亮。

上了幼儿园的我，毫无意外地成为了午睡困难户。中午别的小朋友睡得香甜，而我在床上莫名烦躁。

有一天我不知道从哪里弄来一个签字笔笔盖，正愉悦地玩耍着，试图熬过一个中午。但是人算不如天算，这个笔盖竟然从我上铺的床上直接飞了出去，直接飞到守着大家午睡的老师脚边。我慌了，在老师有所行动之前紧紧闭上了眼睛。

后来发生的事情我选择性遗忘了。唯一记得的是老师从我身边经过的时候我试图装睡又不得其法的由于闭太紧而不停眨巴的双眼。

上了小学的我印象中从来没有午睡。

那时候的我晚睡晚起，经常性迟到。往往是同学们已经快做完课间操，我才溜进学校。放下书包躲在楼梯某处，装作和大家一起回教室。对于这种明目张胆的违纪行为，老师也不怎么管 - 毕竟我在我那所小学成绩优异，且在政治上毫无追求（后来的区三好学生市三号学生都与我无关）。因而很幸运的，我的睡眠问题没有给这一阶段的我带来任何困扰。

上了初中的我开始寄宿生活。

午睡魔咒再度袭来。那种熟悉的，在室友熟睡后辗转反侧的感觉。要命的是初中课业强度巨大，班里学霸，以及我，甚至会把作业中午带回宿舍，抓紧时间写一点。因而在辗转反侧的时候我还多了一层心理负担，因为我不仅没有补充睡眠，也没有多写一点作业。更要命的是晚上，我试图早一点睡，以此弥补中午缺的觉，然而我的室友往往会夜聊。我的内心极其矛盾，一方面我想加入，另一方面我想尝试补觉。最后结果往往是，我的沉默会让室友误认为我睡着了，她们兴高采烈聊完天睡着之后，我依旧辗转反侧。而且，即使要翻身，我也不敢过于夸张，不然就不像是一个“睡着的人”。

就这么三年过去，我的成绩缓缓下降，脑子越来越慢，理综的拖后腿性初见端倪。

高中我换到一所公立中学，重新成为走读生。然而逃不过的，仍是想睡却睡不着的午觉。

走读生无床可睡，我也赶不回家，只能趴在桌上凑合。那时候班上有一个像永动机一样的女生给我留下很深的印象，她不管小课间大课间还是中午，永远不需要睡觉，永远在写卷子或者归纳卷子的错题集。我真是实名羡慕。与此同时我还羡慕另一帮人，他们一旦开始睡午觉，几乎就能睡着，甚至还会打呼噜。而我，没有精神做一中午卷子，也没有办法熟睡到打呼噜，就只能迷迷糊糊半睡半醒地度过午休时间。这种补充精力的过程就像给一个被针扎过的轮胎打气 - 针眼不大但是气永远充不满。

上了大学的我心态已经逐渐平和了，自我接纳的过程是如此漫长。

中午依旧睡不好 - 在下午有课，必须按点起床的情况下。睡不好就睡不好吧，反正那么多年也过来了。印象中只有一次，晚上被室友打游戏的声音吵地实在睡不着，气到裂开。后来那位室友流泪道歉，我没有接受，她也没有改。大四换寝室，蓄谋已久地和这位神仙分开了。那一年的我已经开始践行迄今为止最为简单有效的办法- 晚睡。既然我做不到在周围有人没睡觉的情况下睡着（即使她只会发出很小的声音，但是只要我听见了，或是说只要我知道有人没睡，我就很难睡着），那我就成为那个最晚睡的人。这个办法非常管用。这也得益于当时的大环境，一间六人寝，两个北京人基本不在寝室住，一位喜欢去别人寝室睡，剩下的两位作息都相当规律。总之大四的我睡眠质量竟然还不错。

之后去美帝读研，住2b1b，室友和我睡得都不早，当然我还是基本贯彻了晚睡原则，即使只有两个人，为了睡好也要成为晚睡的那个。

再后来开始工作，经历了独居-与人同住-再次独居，晚睡这个从前的战术性策略已然演变成为一种恶习。并且这个时长不知道是21天的多少倍，一时半会儿恐怕是难以扭转了。

晚睡虽然能极大程度上保证尽快睡着，但顶多只有99%的置信区间。剩下1%是令人闻风丧胆的失眠。失眠失眠，不失则已，一失惊人。一旦失眠，入睡时间往往三四点起跳，甚至能亲眼看到阳光重回窗前。最近的一次失眠是在面挂一家国内互联网公司之后，直接失眠到六点。所幸之后昏睡一上午，算是没有太消耗元气。

所幸工作之后基本上没有午睡这一环节，也就不必面临早年上学期间在学校睡不着，或者周末在家爸妈都在午睡而我无所事事这么折磨人的事情。阿弥陀佛。

不过话说回来，我不是不喜欢睡觉，只是不喜欢是午觉。周末经常睡到中午才起，也是十分酣畅淋漓。
