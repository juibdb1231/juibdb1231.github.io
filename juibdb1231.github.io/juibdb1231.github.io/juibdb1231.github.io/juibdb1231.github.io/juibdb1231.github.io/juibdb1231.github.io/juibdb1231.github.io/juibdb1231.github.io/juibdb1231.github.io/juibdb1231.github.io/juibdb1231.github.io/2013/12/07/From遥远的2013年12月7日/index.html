很远也很近的过往

科技馆之儿童乐园做志愿杂感

儿童乐园，不仅有众多儿童，还有更众多的家长，而仅仅是滑滑梯这一简单项目（其实也不能说简单，因为该滑梯较之于普通滑梯，更长更封闭），通过它不能说世间百态尽收眼底，但起码可窥见一隅。下举数例予以说明。

某男，看似一副正人君子模样，却因为某位母亲纯属无意插了队导致自己儿子晚了一些滑下去，便气血上涌破口大骂，极尽难听之事。本认为这种毫无风度的男子可气可悲可笑可怜，但随后眼见他陪自己的宝贝儿子跑上跑下毫无怨言，上下十多二十次，耐心到极致，不由地怀疑这究竟是不是同一个人。他不是个好男人但是绝对是个好父亲。矛盾？也许吧…人的可怖的多面性再次得到证明。

某女，看似一副高知的模样，却在连问都没问过儿子的情况下与老公争执着，说不允许儿子下去。注意她用的是毫无理由可言的不行不可以不能不允许。最后她败给她老公了，只能看着她儿子下去，然后，一声毫无征兆的“天啊，他掉下去了”把我惊出了一身冷汗。试问这又不是刀山不是火海，没有让你儿子英勇就义，不过是个滑梯，是个游戏，何至于？

家长作为一个施教者，应该随时警醒，做一个规则的捍卫者而非利用者，做一个孩子潜能的挖掘者而非埋没者。我看到太多的家长不管三七二十一把孩子抱上去，然后紧紧拽住，然后大声叫下去吧下去啊，再十分留恋不舍地放手。我十分欣赏的一对年轻夫妻，蹲着问孩子你要不要去试试，得到肯定答复后便悉心指导，但绝不过多干涉，待孩子慢慢摸索着下去后，我听到愉快的笑声，看到年轻的爸爸妈妈眼中满满的爱意和真诚的赞赏。随后，小朋友又玩了几次，满足而欣喜。我问年轻夫妻小孩子几岁啦，答：三岁半。

另外，我十分困惑的一点是为什么无数家长会指着其他其他人对自己的孩子说你看其他小朋友都怎么怎么样所以你也应该怎么怎么样。这是什么逻辑，其他人和你的小孩有半毛钱关系？你既不知道他们的年龄又不知道他们的性格，难道这也是一种比较，一种无聊的，没有任何价值的虚荣？这种常见到令人麻木的现象，害了多少小孩，多少家庭？适当适量的比较会是好的，倘若滥了漫了恐怕就会让结果南辕北辙并且北得比你想象的远。

说说几个印象深刻的孩子吧。小A，在这个喧闹的地方在爸爸的陪伴下耐心摆弄着“手眼协调”（一圈铁丝和一个铁环，使铁环通过但不接触铁圈），非常安静沉稳的女孩子。小B，每次都紧跟在一个小男孩身后下滑，并且在下滑前要大声询问小男孩你到了吗我可以下去了吗数次直到得到笃定答案才敢下去，这种对别人（尤其是异性）的过度依赖在我看来相当可怕，因为毕竟最可靠的人是自己，永远都是这样。小C是个土豪，才上幼儿园大班的他已经有手机电脑和平板，并且可以和比他大十二岁的我几乎无障碍交流，尽管他一直没说清楚爸妈的工作究竟是什么并且不明白志愿者是什么意思而且坚持叫我阿姨不肯改口叫姐姐。

终于比较能明白为什么在国庆六十周年阅兵的时候胡主席在面对一堆高精尖武器装备时表情依旧严肃，却在看到孩子们笑着跑来时露出久违的笑容。孩子是祖国的未来（当然我也没有完全脱离祖国的未来这一广大的范畴），所谓后浪推前浪（不一定是长江啊，黄浦江金沙江都是这样），这是理论意义上的。当我们审视教育，却发现，未来堪忧，前浪危险。教育是一门艺术，它是分时间空间种族地域的，甚而至于是分高低贵贱的。没有普适真理，更无法统一标准。我们曾经历的，以及今后转换角色将经历的，无疑充满巨大悬念与挑战。对待现实，无法将其抬到与理想等高的位置，亦不可将其贬损到不堪。运用理性与智慧，经验与方法，塑造好自己和下一代是我的选择。

好吧，我承认自己是回不去了，否则不会站几个小时就觉得累得遭不住，不会有那么多乱七八糟的想法，更不会深更半夜码那么多字而基本无中心无重点啰啰嗦嗦不知所云。
