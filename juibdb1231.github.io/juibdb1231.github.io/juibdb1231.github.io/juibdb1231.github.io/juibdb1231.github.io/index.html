一次在公交车上关于生而为女生的十万个为什么

坐在摇晃的公交车上，看到路上走着一男一女，女生上身穿着紧身衣下身穿着超短裤，男生上身穿着宽松大T恤下身穿着快到膝盖的休闲短裤。

从什么时候开始女生的服装种类远远多过男生？难道从母系社会的结束，采集这种生存方式被逐步摈弃的时候就已经开始了吗？

除开身体结构的不同，就单说鞋，为什么女鞋种类远多于男鞋？

“女生生来爱美”这话有科学依据吗？还是说仅仅是社会发展过程中逐渐形成的一种人类共识？男生难道就不爱美吗？就算女生真的比男生爱美，是从什么时候开始的？

从女性的角度来看，难道只有男性凝视的存在吗？女性不也照样在凝视女性。况且和言论自由类似，凝视也是自由的，任何社会没有权利捂住人们的双眼，那这个事情的界线又在哪里？

难道只有国内的女性才经历职场性别歧视吗？

国外的女生在她们的国家就没有职业天花板了吗？还是说国外的是挑高的？

我始终坚信男女无法做到真正意义上的平等。我没有做过研究，毫无科学依据，只是凭借目前的认知水平，我认为身体构造上的不同导致的形体差异，力量悬殊，生育角色的差异，是导致这一切的不平等存在且无法消弭的最不可撼动的理由。其他的诸如思维方式，社会期望分工等等比较抽象的差异，均由其衍生。

在我个人的成长经历中，有来自家人的“你这个样子怎么嫁的出去”，有来自男小学课外辅导老师的“女生别去学计算机编程”，有来自亲爹的“要是遇上个机灵小伙子就好了，问题一定能解决”。

很遗憾，我到目前为止没有能够用实际行动对以上言论进行有效回击——我的确没有“嫁出去”，也没能学好计算机编程，甚至于去公寓楼下取网购的床垫，也在“机灵的小伙子”的帮助下才得以把床垫放上小推车。

公交车继续开着，地方很远，要开一个半小时。耳机里放着我最爱的播客，随机波动，那一期是“人的疆域，略大于刘小样的平原”。

主播情绪如此激动，她说“她在刘小样身上看到自己，也看到很多人——内心有一团火在燃烧的人、不甘于现状努力突围的人、不断寻求内在超越的人。这种超越关乎持续的精神追求，关乎人的尊严，关乎我们是否要以成败、以目的论和现实的功利考量来评判人、衡量人、指导人。更进一步，这也关乎在藩篱高筑、界限分明的当下我们对于人的疆域的想象——人为什么不能有跳起来去够的权利？人为什么不能拥有支持性的社会网络？精神追求为什么一定要有物质保障？我们为什么丧失了对主流以外的生活的想象力、语言和行动？”

好像是有点跑题了。

从德云社到上海

从德云社到上海，时间距离一年，空间距离11500公里。

去年暑假腾讯出了款综艺叫德云斗笑社，时值人生空虚加上时间充裕加上出于好奇，在对德云社的了解仅限于听说过郭德纲于谦郭麒麟的基础之上，追完了所有正片节目与衍生花絮，从此在多伦多入坑。之后便以强大的自驱力疯狂恶补所有德云社的有关情况，对德云社的成立到后来艰难曲折的发展，郭德纲本人的种种过往，他的家庭他的徒弟们，德云社常演的不常演的相声，凡此种种都有了深入了解。期间通过各种渠道搜集相关资料，兢兢业业，效率惊人。有一段时间甚至每天听相声三五个小时，即使内容重复也去找同一对或者不同搭档的不同版本，最终熟练掌握传统相声三十段以上，对梁子（黑话，故事梗概的意思）熟稔于心，小曲小调说来就来，唱得虽然难听，但是自娱自乐足够了。更加难能可贵的是始终保持对一个非实用领域求知心，有一回给杨爷暴力安利完头德云社的头九之后被她一句“那头鹤呢”给问住了，于是查漏补缺，编织出了一张更加细密的德云社知识网络。

那阵子对德云社这个团体充满了无限向往，晚上睡觉前还会幻想着阴差阳错下加入这个草莽江湖。故事可能是这样的版本：本人性别男，自年幼起不爱读书，整日昏昏噩噩，惹事生非，家里也实在没有办法，在我接受完九年义务教育之后默许我进入社会接受毒打。之后我发现自己一来没文化二来没技术，只能做一些杂活，勉强填饱肚子。之后误打误撞进了德云社，一开始什么也不会，每天擦桌子扫地，给各位演员打杂，日子过得相当局促。几年过去，慢慢学到了一些相声段子，再加上人又老实，逐渐有了零星的上台机会。呆的时间久了，和这个团伙产生了感情，身上也烙下这个圈子的独有印记。然后有一天和几个差不多同时加入的社会人士一起拜师，有了各自的艺名，有了一群师兄，将来会有一群师弟。周围都是人精，虽然我老实但却不傻，有几个关系铁的哥们儿平时一起吹牛，谁有事了说一声哥几个能帮就帮，有钱出钱有力出力都没有的话出吆喝。

如果说非要从上面这段脑洞里提取出什么，那么可以看出我对一些东西的渴求，其中包括兼具个人和社会认同的职业属性，有区分度的专业技能，身边能够见着面聊得上话的同行，以及生活中无需防备愿意为其两肋插刀的朋友。至于说这些东西是否会求而不得目前尚未可知，时间会给出答案。并且要说明的是我并不是鼓吹知识无用，只是基于当时的情境，对从小到大的线性成长轨迹产生了额外的想象。家里虽然并非大富大贵有权有势，但也从没有过极度困窘生存艰难。在求学路上一次又一次的残酷筛选中跟上了大部队，和很多人一样念书考试，和一部分人一样出国，和其中一部分人一样在毕业后在国外工作。时至今日回想起来，所获取的知识带给我往前走的资本的愉悦感是远胜于实用或精神层面的纯粹愉悦感的。而德云社直接消解掉了前者，当然后者也几乎不复存在。德云社先前被社会规则压制，落于下风，最后又被同样的社会所产出的规则接纳甚至欣赏，这本身就具有一定的讽刺性。

目前德云社发展势头良好，相声综艺电视剧全面开花，团体的头部艺人阵容强大数据喜人，龙字科和鼓曲社的招新工作有条不紊开展，仿佛一代江湖霸主。然盛极必衰，合久必分，未来某一天这一切必将烟消云散。所以很想有机会在德云社走下坡路之前去贡献一份票钱，来感激这一年来给我的远程虚拟陪伴。除了拿不到票的房地产商组织的内部场，以及齁贵的专场，能近距离看到活人的地方就只有小剧场。然而我目前所想象的未来一年落脚地是上海，上海没有德云社小剧场，这令人非常遗憾。尽管现在我已经很久没有完整看过一整个节目了，德云社就仿佛那种无须多言的老伙伴，偶尔刷一下消息，看一些小片段，等着八月份斗笑社第二季上线就好，但是如果有机会去北京或者天津玩，这个票钱是一定要花的，这个约是一定要赴的。

为什么会是上海？为啥要回国？一个庸俗且不堪一击的理由是上海有很多家便利店。我非常喜欢逛超市文具店便利店等等地方，也曾幻想过有朝一日能自己开一家小超市。

抛开上述的玩笑理由，原因之首在于工作签证岌岌可危。其次，在甄嬛传里，花穗和小印子在余答应指使下试图让甄嬛慢性中毒。这像极了我现在的状态——看似每天只是感到深思倦怠，可长此以往，便会状如痴傻。再有，第一次到美国是大二暑假那年去UCLA交换，去之前曾经在上海住了几天。静安寺附近高大的梧桐给我留下了极深的印象，冥冥之中命运之手仿佛推着我去那个城市生活，尽管买不起房。

于是开始给上海的公司投递简历，态度谦卑。昨天和一个国内著名加班公司的hr聊天，hr用非常礼貌的声音问我说，自己听说国外的研究生大多都是两年，为什么我的康奈尔研究生只有一年。我一面心想说你可真厉害，都是从百度听说的吗，一面跟他耐心解释。之后该hr又问到我婚恋状况，并且主动向我介绍他家的工作时间，语气平稳，非常专业。我虽然早就在心里写满拒绝，但最后用同样礼貌平稳且专业的声音告诉他我什么时候有空面试。果然，人若是清高，是因为还不够想要。

疫情之下独居近两年，逐渐意识到自己其实是依靠周围的人活着的，我可能没有自己想像得那么不需要朋友和家人。况且只是换了一个地方工作和生活，只是有了不一样的欣喜和不一样的不如意。自由太过奢侈，能够拥有自己消费得起的自由就已经足够幸运，可问题是地广人稀并不一定意味着自由。看了一期锵锵三人行，对两句话印象很深。一句是王朔的“崩溃就是想起了以前的历次崩溃”，另一句是《地藏经》中“一切众生，未解脱者，性识无定”。或许当一游民也未为不可。

回想起上一篇文章的完工，仿佛是很久远的事。人（我）其实比想象中还要拖延。呵呵。

叶老

叶老是我的初中数学老师，男，四川乐山人，面容沧桑，声音低沉，普通话极其不标准。

叶老的教学相当有特色。十多年后的今天，我依然记得考完试评讲卷子时，叶老会一道一道题挨个讲，并且会让做错这道题的同学站起来“示众”。如果站的次数多了，对自身而言也是一种羞辱。有的同学试图蒙混过关，不在自己做错的题目上站起来，立马会被叶老的一双火眼识破，之后再也不敢偷懒。不知道叶老哪里来的神奇记忆力，以及精准的识人之术。托小学奥数的福，初中数学知识对我难度不大，有段时间上叶老的课我甚至在默写周杰伦的歌词。因此叶老留给我最深的印象并非其高超的教学水平（虽然这是事实），而是一手优秀的粉笔字。叶老的字遒劲有力且娟秀，从书法的角度可以粗略类比瘦金体，从园林的角度神似盘绕着的盆景枝干。叶老握粉笔时只用上手指的一小块面积，书写时轻拈粉笔，刷刷点点行云流水。板书完成后手上、衣服上清清爽爽，不沾一点粉笔灰。

叶老走路的时候略有些跛，夏天的时候能看到他左右两边的小腿不完全相同。然而叶老气场很强，完全不会受到这一点小问题的影响。一次某同学问叶老腿怎么了，叶老轻松回答说被忽悠了。那个同学想必没有恶意，但是叶老的这个回应让我对他的佩服之情更深一层。气场强大的叶老有时也非常温和，当然了温和的对象并不是我，而是我给起了一“朱大”外号的室友（跑一下题，关于这个奇奇怪怪的外号我唯一的印象就是和历史课本某一章的朱德有关，外加该室友年龄排老大，二合一简称朱大）。朱大是一软妹子，英语特牛，天赋极佳，傲视群雄，但是数学相较于英语略略有些偏科，朱大学得一丝不苟，考前仍不免紧张焦灼。叶老看见了，安慰道，没事不用担心有叶老师在。我当时听到了之后的内心真实活动放到今天就是一脸问号的看手机地铁老人。感觉这话从叶老嘴里说出来不仅诡异，而且不符合逻辑，对于考试拿分更是丝毫没有帮助。当年还是太天真，心理上的积极暗示对人所产生的巨大作用我一无所知。

“无知”的我和叶老之间发生的另一件事，在我平平无奇的人生中绝对算是一个突兀的存在。

事情的大背景要从室友曾老三讲起。初中寝室四人一间，除了我和朱大，还有曾老三和老幺。曾老三成绩在班上倒数，好在学习态度端正，也不太招骂。除此之外，来自自贡的曾老三还会时不时给我们带来一些家里人做好抽了真空的自贡冷吃兔和冷吃牛肉。这个真是好东西，真材实料，纯手工制作，不添加任何防腐剂，美味又安心。有时候晚上熄灯了，我们溜下床，开一袋冷吃兔或者冷吃牛肉，打着手电筒享受美食，尽量不被走廊宿管大妈以及楼外巡视大妈发现，紧张刺激的心情给食物增添了风味。有点扯远了。想说的重点是，由于曾老三的这些美食，我们在给曾老三提供各类学习帮助，诸如借笔记讲题之时，也没有什么怨言，宿舍呈现出一派互帮互助的和谐景象。

某日，我在教室里给曾老三讲题，叶老拿着一沓作业进来，让大家帮着发。他也叫了我，但是我确实没有听见也没有注意到。叶老那天心情也不是很好，他看到我没有搭理他，瞬间爆发，生气至极，让我这节课在教室外面听。被赶出教室后，我在外面站了一节课，当时的内心活动不大记得了，不过肯定不是深刻反省并意识到自己错了。数学课之后叶老又让我去他办公室，对我进行思想教育。人真得只能记住部分自己潜意识里想要记住的事情。“被赶出教室”事件发生后，我不记得周围同学们时怎么看我的或者跟我说了什么，也不记得自己当时想不想哭，能记住的是此事惊动了班主任和我妈。两人火速联系，就像处理一个公关事件一样，生怕造成任何负面影响。

负面影响之一就是关于我的“政治前途”——评优，当然也包括我和叶老之间会不会就此结下梁子，我的数学成绩会不会受到影响等等。不知道是幸运还是巧合，这些事情都没有发生。我妈带着我找叶老时，叶老坦承当时他确实脾气没有控制好（此时沉迷相声的我随即联想到朱鹤松的那句你又暴躁了），此事我虽有错也不是百分百都是我的错。后半句显然不是他原话。和我妈交流期间还问过我最近印象深刻的知识点，我非常流畅地回答说是求梯形的面积，叶老面带赞许。

说回曾老三。该事件过去后我和她也还是和之前一样，吃人嘴短。可惜的是曾老三实在学不太动，初三留级，从寝室里搬出去了。四人寝室变成三人，不论是从轮值日还是冷吃兔的角度来讲我都有些难过。

2009年11月，初三的我们经历了一场大事——学校停课，教师静坐抗议待遇不公。学校是私立性质，老师的工作相较于公立学校而言强度大但待遇并不高甚至更低。这种以往只能在电视或书本上见到的劳资纠纷，真实地发生在自己身边，那种人心惶惶的感觉很难形容。事态最严重的时候，我们会在学校上晚自习的途中被要求回寝室避一避。那阵子我们正在准备学校的初升高直升考试，回家自己复习也不是办法。勉强恢复教学后，数学课上的叶老眼神坚定，双拳紧握，我不知道如何准确翻译叶老想传达给我们这群半大不小的初三孩子什么，也不能确定记忆有没有偏差或是夸张，一心备考的我默默打开桌面上的黄色试卷夹，整理错题，查漏补缺。

毕业之后我单独回学校找过叶老，他在学校附近租了房，里面有十数张桌子，椅子倒扣在上面。黑板上仍是那熟悉的粉笔字，没来得及擦掉的板书令人怀念。叶老坐在餐桌边，目光望向远方，全程没有看我，自顾自地说着一些颇深奥的话题，可能也有某些人生哲理。中途叶老没有给我留话口，讲完之后起身送我去地铁站。

还有两次和同学约着去看叶老。一次是叶老请客吃烤鱼。另一次印象颇深，时间是我在康村念书第一学期结束回家过春节的2018年，地点是某栋写字楼里的棋牌桌游娱乐室加私房菜馆，大家集体凑钱平摊费用。我麻将水平菜到几乎可以忽略，于是在桌游组划水，朱大和叶老在麻将桌上厮杀。中途去看了一会儿，大家都比较严肃，高手过招，胜负只在毫厘。吃饭的时候三生石语文老师也拨冗前来，他在最喜欢的“汇报近况”即“你在哪儿念书工作了”环节第一个cue我回答，让我这样一个非常不善于在此种环节表达的人很头大。好在康村比起本科，多少上档次一点，不用担心会像之前那样被语文老师无视掉。磕磕巴巴说罢，我有点陷入恍惚，目光空洞。叶老看到了盯着我说，诶，别发呆了，把我拉回了现实。

同样缺乏逻辑的是整篇往昔回溯。当我断断续续敲下这些混乱的文字之时，愈发感觉到记忆的模糊与无用。无论是当年我被叶老赶出教室还是老师们罢课，款曲之隐晦人性之幽微，其实可以无限放大到权威与服从，民主与反抗等等深刻的层面。然时过境迁，宏大是经不起消解的。将来如果有机会能再和叶老一起吃饭，一起聊聊闲天就好，不要去想意义，毕竟意义的意义又是什么呢。

脑子里的胡思乱想被打断，我很不爽，仿佛拉屎拉到一半被人拎起来。

“对。”

“哦。”

“这是一个梦。”

“梦也要有现实基础。”

她难道被人远程操控了。

她的大脑不会是缸中之脑吧。

2021年1月3号，我和岑姐打了一通微信电话，时长333分钟。

那天在电话里，岑姐聊到自己的工作，同事，老板，任务，聊到自己想跳槽，聊到当前的一些困惑和纠结。我以为岑姐会在家里打电话，但是岑姐告诉我那天香港天气很好，于是出门散步爬山。途中有一段路信号特别差，岑姐的话听着就像电音。试图辨认岑姐在说什么未果的我有些恍惚，脑海里闪过大一升大二暑假，和岑姐一起去大西北旅游的许多画面。

那是2014年8月，我从成都出发，岑姐坐火车到兰州和我会和，至今我还记得见到岑姐时她提着的那个看上去非常沉的旅行皮箱。之后的行程在时隔多年后的今天回想起来依旧令人沉湎。在兰州，我们去夜市一家上过舌尖上的中国的小店铺品尝牛奶鸡蛋醪糟，我俩一个排队一个坐在小桌子边占位；在马子禄牛肉面店，我俩运用同样的一人排队一人占位模式。我第一次知道“兰州牛肉面”才是正统叫法，“兰州拉面”是一个风靡外地的代号而已，除此之外留下深刻记忆的既不是什么“一清二白三红四绿五黄”，也不是“毛细”“二细”“韭叶”等不同面型的称谓，而是排到我时，师傅一大勺滚烫的面加汤盛到碗里，端着两个碗走向座位的我实在受不了高温，直接把碗毫无征兆地撂在一桌陌生人面前缓上一缓，几位陌生人短暂惊讶之后投来理解的目光。

在敦煌，白天我们在街头溜达，晚上则去夜市逛到凌晨一点，回酒店迅速休息几小时，凌晨四五点起床到鸣沙山月牙泉景点排队骑骆驼、看日出。当时我的心情在朋友圈有如下记录：“浓重夜色中一路颠簸着前进，没有体味丝毫黄沙百战穿金甲的雄心，唯感一将功成万骨枯的悲凉。”这里只写到了我的心情，没有关于我分到的那匹在队尾丝毫没有集体意识的骆驼。出发时前面的骆驼都按照指挥起立，它偏不，等到了目的地前面的骆驼听话地蹲下，它依旧站着，工作人员不得不专程从队伍前面跑到我这里来对这叛逆的骆驼进行单独指示。

不知道为何人类仿佛对日出这样一个每天都会发生的自然现象极其沉迷，百看不腻，我也不例外，坐在小沙丘上看着远方天色渐变，天空一点一点亮起，和周围人一样举着相机一同狂拍。太阳升起，新的一天来临，我和岑姐各坐在一小板上从小坡上滑下，体验了一把滑沙的刺激。之后的游览中，艺术细胞发达，擅长舞蹈的岑姐指挥我拍了许多她美丽的照片，其中包含双手舒展前伸并向后高抬一腿等等高难度的优雅动作。

那天晚上，岑姐拉着我去看了以敦煌飞天为主题的演出，很意外的是我竟然没有看睡着。看完演出我们在路边买了蜜瓜，嘱咐老板切开成小块，方便食用。回到酒店后我俩完全没有任何引申含义地吃瓜，酣畅淋漓。敦煌这座城市并不大，两三天的时间我已经大概认识了方向，这对于路痴的我而言难能可贵。

敦煌之行的另一大亮点是莫高窟。去之前对莫高窟有一种没来由的神往，为了保护壁画每个团只能参观几个洞窟，我曾尝试混入正在参观其他洞窟的其他旅行团蹭一波讲解，但是导游的声音极小，每个团员都戴着耳机听，遂无果。接着旅行团带着我们去到了阳关和玉门关。当年王维写下那句“西出阳关无故人”时不会料到那荒芜之处现如今成为了旅游景点。记得我还买了一个所谓“通关文牒”，有点可笑地强行代入古人。时值盛夏，太阳毒辣，黄沙滚烫，没有“朝雨浥轻尘”，没有“西出无故人”，我去到那里的意义可能就只是一边听着导游的讲解一边给写着“阳关故址”的石碑拍照，照片中还有那么几个同样无知无感的像我一样的普通路人。

临走之前我曾许愿今后要再来一次，不经过兰州，而是直飞敦煌。一则是很想弥补一下由于时间有限以至于没能在鸣沙山开敞篷车的遗憾，二来是想重游莫高窟，三来是十分怀念随便买不踩雷的大甜瓜。

信号慢慢变好，电音消失，岑姐告诉我她现在已经到了山顶，打算去买点吃的。电话那头的岑姐用我听不懂的粤语迅速地买好了食物，找了个地方坐下边吃边聊。我问岑姐在公司是说粤语还是英语，答曰都说，和同事基本上讲粤语的。岑姐和家里讲普通话，和我打电话讲四川话，四门语言一点没串。我告诉岑姐自己的英语不咋地，岑姐非常诧异，可能她觉得我在一个讲英语的地方工作语言不会有任何问题。于是乎我和她讲了我用‘medicine’向印度同事解释什么是江中猴菇米稀，以及在被问到为什么来美国念研究生时，用‘I’m learning’草率且错误地翻译内心想表达的“师夷长技以制夷”的故事。

谁能料想，岑姐听罢后感叹：美国竟然能买到江中牌猴菇米稀。

其实这一点也不奇怪，毕竟岑姐在香港学习生活了整整七年，就如同我对岑姐向我描述的香港租房种种感到陌生。这个巨大的世界的不同角落实在是有着太大的差别。

大一暑假的西北之行后，和岑姐的再次相见则是两年多以后的大四。那个时候岑姐到北京实习半年，我的研究生申请也已经差不多准备好了，在这种两个人都有相对充足的时间的情况下自然是要约着一起浪的。不过说实话，我们的玩耍项目也无非是吃饭看电影逛街这老三样，毫无新意。曾经去到岑姐的出租屋参观，房间不大但是十分精致。岑姐依旧喜爱跳舞，房间里摆着舞鞋，还问过我是否知道靠谱的舞蹈老师，我只能报以礼貌又不失尴尬的微笑，不知岑姐为什么会把这个问题抛给没有丝毫艺术气质的我。

最后一次和岑姐一起坐北京地铁，我们看到了美心——一个香港品牌的广告，我问岑姐“美心”用粤语怎么念。之后我去了漂亮国读研，岑姐本科毕业后在香港工作，一晃又是好几年。

最终我们聊了五个多小时，创下了我微信语音聊天的记录。整个过程都很神奇，两人虽然认识了十多年但是一直以来交集有限。我和岑姐这些年在不同的学校，不同的城市，经历不同的事情遇见不同的人，所以这场聊天是强制性地把我从一个很小的安全区域里拽了出来，把自己的许多故事从头讲起，填充细节。一些在我看来理所当然的事情可能岑姐会觉得有点奇特或者说难以理解，反之亦然。整个聊下来感觉到了交织的兴奋和疲惫，以及某种反思 —— 某种意义上讲自己多年来只是在不同的地方过着同样狭隘的生活，可能拥有的对外界的好奇心在一步一步削弱，视角逐步全体向内。解放思想实事求是，伟大的邓小平同志四十多年前的指示对于今天的我仍具备指导价值。

最近读到一本文笔精彩的书，上面说“大地的另一面是梦中的世界，我们则在那个世界的梦中”。此时此刻我用星巴克的马克杯喝着百分之七点五的酒，听到乾坤带里的那句唱词“庄王命文武把冠缨摘去，先饮杯中酒，然后秉烛光”，看到那句“我盯着杯中晃动的酒，朦胧地感到，物质间有不可思议的流转，也许多年前穿过的那场大雾，经过长久的飘荡、流淌和贮藏，最终成为酒，盈盈在这杯中，构成我此刻的熏然”。这些看似毫无关系的语句，因为酒联系到了一起，我和岑姐的两个看似没有交集的人生，因为一次旅行，一场通话联系到了一起。

现如今我早已不记得四年前北京地铁上，岑姐告诉我的“美心”的粤语发音，美心的流心奶黄月饼倒是吃了好几次，这个东西美国也有卖，就像江中牌猴菇米稀一样。

消失在垃圾间后

自打搬到这里，距离垃圾间是最近的一次，不用七怪八拐折上几折，也不用下楼梯出门走向两个硕大无比的装着小区所有人垃圾的大箱子。

我所拥有的垃圾袋分三种，一是超市赠品，二是专门适配本市一种低矮且横截面为长方形的米色垃圾桶的，三是纯黑的，体积大得有些突兀，在影视剧里可以用来装尸体的那种。

超市赠品垃圾袋已经全部用完了，毕竟塑料制品不环保，尽量还是使用布袋子。之前买了家附近名叫“农场男孩”的超市的自有品牌牛奶-“农场男孩”牛奶，瓶子回收，两块钱一个。晚上拎着四个空瓶子，里面最初装着巧克力奶，含脂肪1%牛奶，含脂肪2%牛奶，以及蛋酒。叮铃哐啷换回八块，淋了点雨，手臂隐隐作痛。

专门适配本市垃圾桶的垃圾袋，是在一个上门做房屋安全检查的大哥们赠送我印有本市logo的垃圾桶之后，去超市偶然看见的。那天两位大哥上门，尽管事先有通知，独居的我还是有点害怕。和两名身强体壮的陌生男子共处一室的短短的几分钟，我的视线始终没有离开过菜刀。此外抽屉里用厨房纸包着的换下来的呲了的刀片必要的时候也能用来防身。

说起来，这并不是第一次和陌生男子共处一室。搬来这栋公寓的一年里，大概报过三次物业维修。第一次是没有弄明白空调面板的温度是如何调整的，物业老头极富耐心地像幼师一般教导于我。第二次是头发堵住了洗澡池子，还是那个老头，给我推荐亚马逊的爆款蓝色海星硅胶地漏，用物理方法解决了这一问题。第三次是烟雾报警器以每隔三十秒的频率发出一声尖锐的滴滴声，老头扛着梯子进门，给烟雾报警器换了电池，坚持目测三分钟确保没有问题之后才从梯子上撤下。临走前，老头主动帮我清理了烘干机槽里以及烘干机上方排风口的脏东西，说是会影响烘干效率。

我连连道谢，心里有百分之八十认定老头是好人，另外百分之二十担心老头是禽兽。不过没有关系，我有五把菜刀，十八子作出品，不说削铁如泥也比走大辙的冰糖胡虏老先生的弟子更能保镖。

至于那个纯黑的超大号垃圾袋，买的时候纯粹是因为没有仔细看型号。它不仅不适合我屋子，也不适合那个垃圾间。当它被勉强塞进运输管道，消失在垃圾间之后，天知道它会经历些什么。周围要么是超市出身的垃圾袋，要么是适配印有本市logo的垃圾桶的定制款，身为一个黑黢黢的傻大个，长得丑还碍人道，不可避免地会遭白眼受欺负。

但是它并不觉得自己做错了什么，它坚信天生我材必有用。事实上它的用处不在市区高楼，而在大别墅中。就算不是大别墅，也得是像局长曾经佛罗里达的家那种。俗话说什么垃圾桶配什么垃圾袋，经常学着司机师傅扪心自问到底知不知道自己的定位，总归是没有坏处的。

想当初还算太平年间，去佛罗里达找过局长，去华盛顿和迪神看过樱花，去旧金山和翘楚在金门大桥顶着嗖嗖的大风骑过自行车。现如今局长已回国发展，而我一边哀嚎公司药丸一边默默希望生活能给我来一刀痛快的（非常看不起遇事拖泥带水黏黏糊糊的自己）。迪神的世界已经离我相当遥远了，有机会的话想当面对她说一句苟富贵。这句话的重点其实是后半句，故意隐去显得削微委婉一点。翘楚最近从男人身上花了时间也体会到了快乐委屈愤怒等多重情绪，不知为何竟会找我聊上几场，让我产生了一种能开情感类播客的虚无飘渺的自信，没吃过猪肉好歹也是见过猪跑的。

瞎扯到这，我吃完了本打算明天当早饭的黄油巧克力饼干，很撑，一大波脂肪正在连夜赶来。空盒子会被扔进垃圾袋里，那个大黑个子还没被装满。等过几天它被我赶出家门，发配垃圾间之时，有没有可能稍带着我一起消失。

其实答案是肯定的。我仔细查看过那个垃圾间，假设有人藏匿其中，并不会被开门的人瞬间发现。而他可以利用这个短暂的时间差行凶，一击致命，悄无声息。

同样是十周年，孟鹤堂周九良从籍籍无名走到了如今的鲜花簇拥。很羡慕这样的人，能够同时拥有热爱的事业和一起并肩往前走的搭档，不知道要攒上多少运气才能实现。希望德云社的郭于岳越饼四以及现在名气稍逊的大白萝卜小黑土豆，都能一起说相声说到老。

在花费了很大的力气对自己目前的生活逐步接受之后，一个之前已经刻意忽略的选项施施然飘到了面前。就好比一位故人，已经说好要西辞黄鹤楼，你收拾好心情逐渐接受了它将远去这个事实；然而他并没有在烟花三月去往扬州，反倒折回来问说你还记得我吗，我们一起出去吃饭吧。这一下，让之前付出的，在其他人看来微不足道但是对你来讲很不容易的所有努力，都付之一炬，就像乔英子搭好的乐高被童文洁一巴掌撂到地下一样。

想送自己一句矫情。不过矫情总比憋死自己要好。

众生相

刘师傅

最近迷上了相声，《学哑语》中那句“从小一起长起来的”让我想到了刘师傅。刘师傅年纪不大，比我小将近半年，和我同级，标准成都人。

小学三年级的暑假，我被拎去一个奥数课外辅导班，老师是一陈姓老太。班里有这么一号人物，脑瓜子灵，每次考试成绩特好，此人便是刘师傅。并且听闻刘师傅曾经骨折也没有耽误上课，颇得陈老太赞赏。后来和刘师傅慢慢熟悉起来，一起在班上度过三年，稳坐第二第三把交椅，对于第一名那位大神，我俩难以望其项背。

六年级那个暑假，我和刘师傅一起去青少年宫集训，每天仿佛有用不完的精力。之后我俩一起被选上参加华罗庚全国邀请赛，在宿舍打牌撒疯，给教练起外号学教练讲话，边学边录边笑得抽搐，而同住的一位初一小姐姐在我们吵闹时从不参与，安安静静捧着余秋雨的文化苦旅，五年以后成为了成都市高考文科状元。比赛完大家一起去香港澳门玩耍，我们被去赌场过瘾的教练欺骗，乖乖待在酒店，但也丝毫闲不住，在床上乱蹦学四大天王唱歌，饿到不行了就用纸杯泡酸辣粉，拿一次性牙刷手柄那头当筷子使。总之是留下了一段难忘的回忆。

说句题外话，今天回看小学这段奥数经历，有以下三点感受：

  1. 当范围足够小且限定条件足够多时，名列前矛不是一件难事
  2. 小学奥数的本质是算术

上了小学，每天吃晚饭的时候会见着Z老头，上了初中，每周从学校回来会见着Z老头，上了大学，每学期从学校回来会见着Z老头。时间一年一年过去，我越来越了解Z老头，也越来越不喜欢他。

Z老头极其爱面子。春节全家聚餐，作为长子的他享受着那种被奉若上宾的感觉——一定要别人打电话来邀请他，一定要在饭桌上被周围人敬酒叫着大哥。Z老头性格孤僻，朋友本就不多，但还是不愿意主动联系，每次都是别人打电话邀请，他才出去同老伙计们一起打麻将。Z老头有时把我妈买的新鲜水果放着不吃，水果慢慢不新鲜甚至烂掉，再在我妈下次去看他的时候拉着一起吃；在我妈给他送菜的时候处处挑剔，甚至莫名发脾气甩脸色，活脱脱一个苏大强。

有一回Z老头生病，我和我妈送他去医院输液。病房里其他人都在安静休息，他却一个劲讲话，这么说都不听，完全不考虑这么做会不会影响到别人。回家之后我实在是没崩住大哭一场，我很害怕自己老了之后会变成这样，自以为是，理所应当地做影响别人并且让自己的小孩自己的孙辈如此难受的事情。我妈赶紧安慰我说不会的，她老了以后不会这样，我老了以后也不会这样。可是谁知道呢。每个人潜意识里都是按照自己的想法生活，每个人的行为都基于自己对世界的认知。如果这个认知本身就是有问题的，而你没有一个兼收并蓄的心态，陷在自己的世界里，又怎么可能去调整呢。

Z老头没有办法像拦着我妈一样，拦着我去寄宿学校，拦着我去北京，拦着我去美国，拦着我去加拿大，尽管他不止一次地说，唉走那么远干什么。我没有兄弟姐妹，Z老头没有机会重男轻女，我可以忽略他所有的潜台词。

前段时间我妈带Z老头去医院拿药，抽空和我视频。Z老头出现，问我打算什么时候回去。我心想如果被公司裁掉了没了身份当然就会回去。但我明白他不是这个意思。我可以想象他那时穿着我之前给他买的斯凯奇，舒适柔软，我也可以想象挂断电话之后他持续不断的絮叨，脸上一副执拗表情。

只是和当年推着满头自来卷的我出门的那个人相比，更瘦，走路有点晃，背驼着。Z老头房间的气味早就在我记忆中消散，垂下来的尼龙拉绳，咔哒一声关掉了那盏床头灯。
