梦到醒不来的梦

我来到心理咨询室，对面坐着一位挂着职业微笑的中年女人。

“说说吧你又梦见什么了。”

这不是我第一次来这里做心理咨询，上次来的时候这里还没有装修，仍旧保留着上个世纪老诊室的朴素风格。医生是位年纪很大的老爷爷，慈眉善目。相比而言，眼前的这位虽然介绍栏里写着“经验颇丰”，在我看来却还是生涩。

“你在回忆什么吗？”

脑子里的胡思乱想被打断，我很不爽，仿佛拉屎拉到一半被人拎起来。

不过转念一想，这位名医按时长收费，白白耽误时间确实不划算。

“我梦到了成都大杂院。”

“成都？你是成都人？”

“对。”

“我也是。”

“哦。”

“成都据我所知没有大杂院。”

“这是一个梦。”

“梦也要有现实基础。”

说真的，那一瞬间我很想拍案而起拂袖而去。但考虑到那笔不菲的咨询费退不了，还是忍了。

“我不想讨论梦到底需不需要现实基础，这个问题不归我管。我只想告诉你我梦见了什么。”

“好的，开始吧。”面前这位身着白衣的机器人翻开硕大一个笔记本，从胸前的口袋里拿出钢笔，打开，笔盖在桌上滚动三秒后静止。

“我在这个大杂院里租房住。房东男，就职于建设银行，职位不详，曾参与一款银行创新产品的设计，宣传册上有一张他和外国人开会的照片。我刚搬进来的时候，房间里有部分没带走的保湿产品，还有两箱零食，里面是各种口味的散装格力高。有一天老娘来看我，说趁我上班前给我做饭。她正忙着，我抬头看见房顶灯上有一蠕动的黑黄条纹相间的长虫，体积很大，塑料质感。后来发现这不是虫，而是一个机械小玩具。我用粉色的塑料晾衣杆把这个小玩意儿打下来，之后它在窗台上一蹦一跳。周末我把旧床搬到门卫老头住的那间屋子里，老头是新人，他的上一任干了几十年。我的房间门平时不锁，大家可以随时来串门。隔壁住的两个妹子拿来一个灰色充气垫，方形，相当厚，她们说想玩个类似从高处往下跳的游戏。”

“结束了？”

“嗯。”

中年女人推了推眼镜，面露困惑。我瞥见她本子上记下的寥寥几笔，竟然有一种计谋得逞的得意，毕竟除了我，没有人能够窥探到我的梦。对于这个私有财产我享有无限次篡改权。

“这个梦有点胡来了，一点规律也没有。”

“难道你觉得梦必须做成博士论文那样严谨吗？”

“要不你换一个吧。”

这可是你说的，听了可千万别后悔。我身体前倾，拿出了战斗的姿态。

中年女人把笔记本向后翻过一页，左手手掌使劲压了压，右手捏紧钢笔，微微咬唇，表情严肃。

“我在一所名字未知的学校念书，学校在欧洲某国，可能是德国。操场上，大家正在上体育课，一个人要来追杀我，整个校园仿佛只有我被投下了战争的阴影，和操场上其他人的宁静的欢乐共存。场景迅速切换，我瞬间身处我妈学校教学楼五楼走廊尽头的女洗手间的洗手池边。一男一女分别在一左一右两个洗手池，我正试图杀人。手法非常简单，打开水龙头，用源源不断的流水将二人淹死，杀人不见血，干干净净。谁知女生水性极佳，在充足的水流中缓缓抬起头。害怕被发现正在实施杀人计划的我赶紧装作洗手，并关掉了女生那边的水龙头。我定了定心神，告诉女生我留在水龙头上的指纹是由于我洗手而不是放水杀人。女生没有一丝怀疑，和我一同看向对面那个仍在被水冲刷的男生。我俩谁也没有去救他。洗手池消失，欧洲学校重现。在躲避追杀的路上，我经过很窄仅供一人通过的楼梯，来到一个平台，穿过平台，从平台另一头离开，走出这栋楼，也摆脱了杀手。辨认不清方位的我来到一家奶茶店，店里的抹茶麻薯肉松装在奶茶杯里。我点了最普通的珍珠奶茶，前面还有一个人，我得等着。”

“所以说，你最后用一杯奶茶收买了杀手。”中年女人露出一副恍然大悟的表情。

“我已经说了我摆脱他了。”如果说刚开始的我尚且还有怒火，此时我已然看开，不与傻子论短长，更何况这傻子还是收了我钱的著名心理咨询师。

“不可能的，你只是以为自己摆脱掉了。”之前面无表情的脸上竟露出一丝狡黠。

“你是不是总想从我这个被杀与杀人兼备的梦里总结出点什么，然后就能写出诊断书开出处方药了。”这个指控可以说是毫无根据但我充满自信。

“我想我们还是继续说别的梦吧”。崭新的一页翻开，说话的声音有些迫不及待。

“迈着六亲不认的步伐，我是一只濒临失忆的熊猫，我在回国的飞机上睡了一觉。在卡尔加里，我似乎记得那些邻居们，哦不对，应该在多伦多，或许是华盛顿。不不不肯定不是华盛顿，因为我被要求考雅思，是的，熊猫要在加拿大呆着，是必须要考雅思的。“

“打住，卡尔加里的熊猫我之前去看过，新闻里说现在它们被接回国了，因为加拿大的竹子不够吃。”

我没有理会这莫名其妙的打岔。“回国后我在家楼下火锅店又碰见了我的两个老邻居，他们在清汤锅里涮箭竹。后来那家火锅店和之前这个地段的许多店一样，垮了。”

“这两只熊猫去了哪里？大杂院吗？”

我惊讶于这个中年女人的求知欲。熊猫会去大杂院？你以为是拍还珠格格第一部吗。

“没。它俩想爬上IFS，一个中途打滑摔死了，另外一个现在还一直趴楼顶。”满嘴跑火车我也是会的。

对方一句话不说，眉头微蹙。

见此情景，我索性把脑子里的梦一股脑全说出来。

“我跟在一个很作的富贵老太太身边，她住的房间床特别高，跳下来贼费劲。地毯很软，地毯毛也很高。我和老太太一起下楼，木质的楼梯吱吱呀呀，必须扶好旁边镶着的栏杆。楼下看到一小姑娘抱着俩一模一样的狗子，仿佛闪灵两姐妹。老太太想逗逗狗子，小姑娘和身边的男子看到之后转过来。男子和另一个小男孩开始给我们表演魔术。还没等表演结束，我告诉老太太我要去找我妈。她问我我妈在哪儿为啥要去找她，我说我妈在搜狐大厦宾馆住着，我得找她取充电宝给手机充一下电，事实上我并不只是为了充电。临走前我绕到变魔术的男子后面想看看他在耍什么花招，不幸被男子识破了，他带着小男孩将身体转了过来。我来到搜狐大厦宾馆，走楼梯，和老太太住那地方一样也是木质的。我像是在一个圆筒里盘旋上升，走着走着楼梯就下不去脚了。于是我只能离开楼梯，从旁边开门进到该层大厅中，走在通道里，身后是一个提着旅行箱走路匆忙的商务男。要到门口了我发觉我没有房卡打不开门，遂让商务男用他的卡刷开。进到大厅之后商务男立刻开始social，我径直去往我妈房间。来到一个深色系的走廊，右手边是一排客房，左手边是类似机场登机口处的透明落地窗，窗外停着私人飞机，一间房对应一架，还有工作人员守在旁边，飞机星星点点的亮光在浓厚的夜色里十分突出。每间客房外挂着一个计时器，回想起有人告诉过我这个是用来监控房间里的人的消费频率的，具体来说若计时器到点了还没有任何消费，前台就会提醒房间里的人。我给我妈打电话，刚一接起她就用一种兴奋的语气说她到香港了。我想到之前她跟我提过要去香港但我忘了，瞬间委屈且有些想哭。她似乎听出来了问我怎么了，我说我在搜狐大厦宾馆想找你。我妈先是给我一顿安慰，接着很严肃地说，你想来找我是不是想走捷径，是不是觉得和我坐下来喝杯水，我吩咐下去，你遇到的麻烦就都能解决了。”

我喝了口水，内心做好了被吐槽是不是看过不少类似知音的狗血杂志，或者类似老娘舅的调解类节目。

然而对方依旧沉默，只是在不停咬指甲。

“我很久没有在梦中如此愤怒了，出离愤怒。过年回家，Z老头骑着白色摩托车来接我，行至西门车站老实外对面，老头逆行转弯掉头，摩托车几乎横贴地面，全然不顾机动车道上高速行驶的车流，为了和车流同方向甚至高速倒骑摩托车。受到高度惊吓之后的我出离愤怒，回到家一摔门就进了自己房间。”

“小学数学老师让我把收好的卷子送到楼上的窗台边，卷子都是芹菜做的，很艰难摞起来了。然而外面下雨，风又吹落了几根芹菜，该老师瞬间发火。我下楼参加考试，发现自己没卷子，又专程上楼一趟找她要，没想到她极其和蔼，强压怒火，关切待我。”

“我成为了德云社的一员，某日准备内搭暗紫红色打底裤袜，外面穿一条极其肥大的卡其色潮裤，袜子红底带黄色花纹，搭配红色马丁靴去参加活动。我一边换衣服一边直呼郭德纲大名，内心想师父要在旁边看着可别被其他师兄弟笑话。辛老头出门打算先到车里等我，我从看到分针指到3时的从容不迫毫不慌乱，到指向5，再到指向6，惊觉要迟到了的我去学校已然来不及，遂直接赶往活动现场。次日，于谦和他老婆搬家，我去串门。他家的地理位置好到爆，能从窗外看到文翁石室的苍松翠柏。两人收拾东西很潇洒，留了不少诸如暖水瓶这样的旧物件。因为家里有阿姨善后，所以并不需要一次性搬走所有东西。”

我将杯子里剩下的水一饮而尽，说了那么多实在是口干舌燥。但是对面的中年女人纹丝不动，包括她的耳环。

她是懵住了吗。

她难道被人远程操控了。

她的大脑不会是缸中之脑吧。

脑中闪过无数个念头的同时，或许是喝了太多水，我起身去上厕所。打开诊室的门，门外候诊区热闹非凡。一位西装革履提着公文包的男子一边玩玩具虫一边吃格力高，一个佝偻的老头手握奶茶杯吸着抹茶麻薯肉松，两个小女孩牵着两只狗擦拭两把水枪，一个穿大褂的小男孩趴在熊猫背上写数学卷子。他们是有限的，又似乎是无穷的，越过一个还有一个，穿过一波还有一波，永远抵达不了厕所。

熟悉的闹铃声响起，我睡眼惺忪地起床。目光依次扫过书桌上的心理咨询师胸牌，钢笔，硕大的笔记本，以及今日出诊预约，该患者将向我讲述他那些梦到醒不来的梦。
